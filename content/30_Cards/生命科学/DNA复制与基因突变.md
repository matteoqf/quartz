# DNA复制与基因突变

## 1. DNA复制的基本原理

### 1.1 DNA双螺旋结构与复制起点

|| 特征 | 生物学意义 |
|---|---|---|
| 双螺旋结构 | 两条反向平行链 | 碱基互补配对 (A=T, G≡C) |
| 复制起点 (Origin) | oriC (大肠杆菌) / ARS (真核) | 复制起始位点 |
| 复制方向 | 5'→3' | 新链合成方向 |
| 半保留复制 | 每条链作为新链模板 | 遗传稳定性保证 |

### 1.2 DNA聚合酶的特性

| 酶 | 功能 | 方向 | 特点 |
|---|---|---|---|
| Pol III (原核) | 主要复制酶 | 5'→3' 合成 | 高持续合成能力 |
| Pol δ/ε (真核) | 前导链/后随链 | 5'→3' 合成 | 校读功能 |
| Pol α | 引发酶 | 5'→3' 合成 | 合成RNA引物 |
| 端粒酶 (Telomerase) | 端粒延伸 | 5'→3' 合成 | 解决末端复制问题 |

### 1.3 DNA聚合酶的核心特性

```
高保真性：选择正确碱基 (错误率 10⁻⁵ - 10⁻⁶)
校读功能：3'→5' 外切酶活性 (错误率降至 10⁻⁷ - 10⁻⁸)
持续合成能力：不易从模板解离
```

## 2. DNA复制过程

### 2.1 原核生物复制 (大肠杆菌)

```
Initiation (起始)
├── DnaA 蛋白结合 oriC
├── DNA 解旋酶 (DnaB) 打开双链
├── 单链结合蛋白 (SSB) 稳定单链
└── 引物酶 (DnaG) 合成RNA引物

Elongation (延伸)
├── Pol III 沿模板 5'→3' 合成新链
├── 前导链：连续合成
├── 后随链：冈崎片段 (Okazaki fragments)
└── DNA 连接酶 (Ligase) 连接片段

Termination (终止)
├── Ter 序列结合 Tus 蛋白
└── 复制叉在此终止
```

### 2.2 真核生物复制特点

| 阶段 | 原核生物 | 真核生物 |
|---|---|---|
| 复制起点 | 单一起点 | 多起点 (每染色体数百至数千) |
| 复制速度 | ~1000 nt/s | ~50 nt/s (较慢) |
| 复制叉 | 双向 | 双向 |
| 末端问题 | 无 (环状染色体) | 端粒酶解决 |
| 组蛋白 | 无 | 新合成组蛋白与DNA组装 |

### 2.3 前导链与后随链合成

```
5'─────────────────────────────────────────────────3'
   ↑                    ↑                    ↑
DNA pol III         DNA pol III          DNA pol III
   │                    │                    │
   └────────────────────┴────────────────────┘
                    复制方向

前导链 (Leading strand):
  - 解旋方向相同
  - RNA引物 → 连续合成
  - 单一引物

后随链 (Lagging strand):
  - 解旋方向相反
  - 多个RNA引物 → 多个冈崎片段
  - 引物去除 → DNA连接
```

## 3. DNA损伤与修复

### 3.1 DNA损伤类型

| 损伤类型 | 原因 | 结果 |
|---|---|---|
| 碱基错配 (Mismatch) | 复制错误 | 点突变 |
| 碱基脱氨基 (Deamination) | 自发/化学因素 | C→U, A→Hypoxanthine |
| 碱基脱嘌呤 (Depurination) | 自发/酸/热 | 无碱基位点 (AP位点) |
| 嘧啶二聚体 (Pyrimidine dimers) | UV辐射 | 交联阻断复制 |
| 烷基化 (Alkylation) | 致癌物 | 碱基修饰 |
| 氧化损伤 (Oxidative damage) | ROS | 8-oxoG 等 |
| 双链断裂 (DSB) | 电离辐射/复制压力 | 染色体断裂 |

### 3.2 DNA修复机制

```
DNA损伤
    │
    ├─── 错配修复 (Mismatch Repair, MMR)
    │        识别：复制错误、新合成链
    │        纠正：切除错误片段、重合成
    │        结果：错误率降至 10⁻¹⁰
    │
    ├─── 碱基切除修复 (Base Excision Repair, BER)
    │        识别：受损碱基 (DNA glycosylase)
    │        纠正：切除碱基 → AP位点 → 切除核苷酸 → 重合成
    │        适用：小损伤 (氧化、脱氨基)
    │
    ├─── 核苷酸切除修复 (Nucleotide Excision Repair, NER)
    │        识别：螺旋扭曲 (XPC复合物)
    │        纠正：切除24-32nt片段 → 重合成
    │        适用：嘧啶二聚体、大体积加合物
    │        疾病：色素性干皮病 (XP)
    │
    ├─── 双链断裂修复 (DSB Repair)
    │        ├── 同源重组 (HR) - 需同源模板 (S/G2期)
    │        └── 非同源末端连接 (NHEJ) - 快速但易错
    │
    └─── 直接回复修复
             光复活 (Photoreactivation) - 光复活酶逆转嘧啶二聚体
```

### 3.3 修复缺陷与疾病

| 疾病 | 缺陷 | 临床表现 |
|---|---|---|
| 色素性干皮病 (XP) | NER缺陷 | UV敏感、皮肤癌早发 |
| 遗传性非息肉病性结直肠癌 (HNPCC) | MMR缺陷 | 结直肠癌、子宫内膜癌 |
| 布卢姆综合征 (Bloom syndrome) | RecQ解旋酶 | 染色体不稳定、免疫缺陷 |
| 范可尼贫血 (Fanconi anemia) | DNA交联修复 | 骨髓衰竭、畸形、癌症 |

## 4. 基因突变的类型与机制

### 4.1 突变分类

| 分类标准 | 类型 | 定义 | 示例 |
|---|---|---|---|
| 依据结构 | 点突变 | 单个碱基改变 | 碱基替换 |
| | 插入/缺失 (InDel) | 碱基插入或缺失 | 移码突变 |
| | 大片段突变 | 染色体大片段改变 | 缺失、重复、倒位、易位 |
| 依据性质 | 碱基替换 | 嘌呤→嘌呤 (转换) / 嘌呤→嘧啶 (颠换) | A→G (转换) / A→T (颠换) |
| 依据功能 | 同义突变 (Silent) | 密码子改变但编码相同氨基酸 | UCC→UCG (Ser) |
| | 错义突变 (Missense) | 编码不同氨基酸 | 镰刀型贫血 (Glu→Val) |
| | 无义突变 (Nonsense) | 编码终止密码子 | 提前终止 |
| | 移码突变 (Frameshift) | 插入/缺失非3倍数 | 蛋白截短 |
| 依据来源 | 自发突变 | 复制错误、环境因素 | 复制错误率 10⁻⁸ |
| | 诱发突变 | 诱变剂处理 | 物理/化学诱变 |

### 4.2 点突变的分子效应

```
DNA:  5'-ATG CCC GAA TAA-3'  (原始)
mRNA: 5'-AUG CCC GAA UAA-3'  (Met-Pro-Glu-STOP)

突变类型:
同义突变:
  DNA:  5'-ATG CCC GAG TAA-3'
  mRNA: 5'-AUG CCC GAG UAA-3'  (同一氨基酸 Glu)

错义突变:
  DNA:  5'-ATG CCC GTG TAA-3'
  mRNA: 5'-AUG CCC GUG UAA-3'  (Pro→Val, 可能影响功能)

无义突变:
  DNA:  5'-ATG CCC TAG TAA-3'
  mRNA: 5'-AUG CCC UAG UAA-3'  (终止于Pro, 蛋白截短)

移码突变 (缺失G):
  DNA:  5'-ATG CCC EAT AA-3'  (缺失G)
  mRNA: 5'-AUG CCC GAU AA-3'  (框移, 完全不同蛋白)
```

### 4.3 突变热点与诱变机制

| 诱变剂 | 机制 | 典型产物 |
|---|---|---|
| 5-溴尿嘧啶 (BU) | 碱基类似物 → A:T→G:C 转换 | 转换突变 |
| 亚硝酸 (HNO₂) | 脱氨基作用 | C→U, A→Hypoxanthine |
| 烷基化剂 (EMS) | 鸟嘌呤烷基化 → G→T 颠换 | 颠换突变 |
| 碱基类似物 | 取代正常碱基 | 多种突变 |
| 嵌合剂 | 双功能基团交联DNA | 交联/断裂 |

### 4.4 突变热点形成机制

- **自发脱氨位点**：甲基化胞嘧啶 (5-mC) → 脱氨基 → 胸腺嘧啶 → C→T 转换
- **复制错误位点**：微卫星/重复序列 → 滑链错配
- **化学诱变位点**：烷基化鸟嘌呤 (O⁶-烷基G) → G→T 颠换

## 5. 突变与进化/疾病

### 5.1 突变的功能后果

```
突变对蛋白功能的影响:

功能获得 (Gain-of-function)
  - 新增结构域/功能
  - 组成型激活
  - 显性负效应 (dominant negative)
  示例：致癌基因 Ras 突变 → 持续激活信号

功能丧失 (Loss-of-function)
  - 蛋白截短/缺失
  - 催化活性丧失
  - 单倍体不足 (haploinsufficiency)
  示例：抑癌基因 p53 突变 → 失去细胞周期控制

显性负效应 (Dominant negative)
  - 突变蛋白干扰正常蛋白功能
  - 常见于转录因子/多聚体蛋白
  示例：突变p53抑制野生型p53功能

条件特异性突变
  - 温度敏感突变 (ts mutants)
  - 组织特异性突变
```

### 5.2 突变与癌症

| 基因类型 | 突变特征 | 致癌机制 |
|---|---|---|
| 抑癌基因 (TSG) | 失活突变 (Lof) | 失去增殖抑制/ DNA修复 |
| 癌基因 (Oncogene) | 激活突变 (Gof) | 持续增殖信号 |
| DNA修复基因 | 失活突变 | 基因组不稳定增加 |
| 凋亡基因 | 失活突变 | 逃避程序性死亡 |
| 代谢基因 | 多态性/激活突变 | 改变细胞代谢 |

### 5.3 典型癌症相关基因突变

| 基因 | 正常功能 | 突变后果 |
|---|---|---|
| TP53 | 基因组监护、凋亡 | 多种癌症 (50%+) |
| BRCA1/2 | DNA双链断裂修复 | 乳腺癌、卵巢癌 |
| APC | Wnt信号抑制 | 结直肠癌 (家族性) |
| KRAS | Ras/MAPK信号 | 胰腺癌、结直肠癌 |
| BCR-ABL | 酪氨酸激酶 | 慢性髓性白血病 (易位) |

## 6. 基因突变的检测方法

### 6.1 突变检测技术

| 方法 | 原理 | 检测范围 | 灵敏度 |
|---|---|---|---|
| Sanger测序 | ddNTP链终止 | 单碱基分辨率 | ~10-15% |
| NGS (下一代测序) | 高通量并行测序 | 全基因组/外显子 | ~1-5% |
| PCR-SSCP | 构象改变迁移率 | 已知突变筛查 | ~5-10% |
| RFLP | 限制酶切位点改变 | 已知突变检测 | ~10% |
| ASO杂交 | 探针杂交 | 等位基因特异性 | 高 |
| SNP芯片 | 探针杂交 | SNP分型 | 种系突变 |

### 6.2 突变注释与预测

```
突变临床意义分类 (ACMG指南):

pathogenic (致病的)
  - 明确致病

likely pathogenic (可能致病的)
  - 强烈证据支持致病

VUS - Variant of Uncertain Significance (意义未明)
  - 证据不足以确定

likely benign (可能良性)
  - 强烈证据支持良性

benign (良性的)
  - 明确良性
```

## 7. DNA复制保真性与进化

### 7.1 复制保真性保障

```
复制错误率: ~10⁻⁸ 每碱基每轮复制

三层保障:
1. DNA聚合酶选择正确碱基 (错配率 ~10⁻⁵)
2. 聚合酶校读功能 (错配率 ~10⁻⁷ - 10⁻⁸)
3. 错配修复 (MMR) 系统 (最终错配率 ~10⁻¹⁰)

意义:
- 保证遗传稳定性
- 足够低变异率维持物种特征
- 足够高变异率提供进化原料
```

### 7.2 中性理论与进化

| 理论 | 核心观点 | 突变类型 |
|---|---|---|
| 达尔文选择理论 | 大多数变异受选择 | 有害变异被清除 |
| 中性理论 (Kimura) | 大多数突变是中性/近中性 | 不受选择影响 |
| 漂变理论 | 基因频率随机变化 | 遗传漂变 |

## 8. 核心要点速记

1. **半保留复制**：每条链作为新链模板，保证遗传信息完整传递
2. **前导链vs后随链**：前导链连续合成，后随链以冈崎片段方式不连续合成
3. **DNA聚合酶特性**：5'→3'合成活性，3'→5'校读活性，高保真性
4. **错配修复**：识别新合成链，切除错误片段，保证最终错误率~10⁻¹⁰
5. **基因突变类型**：点突变(同义/错义/无义)、插入缺失、移码突变、大片段突变
6. **突变后果**：取决于突变位置（蛋白结构/调控元件）、氨基酸性质改变
7. **DNA损伤修复**：NER(大损伤)、BER(小损伤)、MMR(复制错误)、HR/NHEJ(双链断裂)
8. **癌症与突变**：癌基因激活(+)、抑癌基因失活(-)共同促进肿瘤发生
9. **诱变剂作用机制**：碱基类似物、烷基化剂、嵌合剂等通过不同机制引起突变
10. **复制保真性**：聚合酶选择→校读→错配修复，三层保障将错误率降至10⁻¹⁰
