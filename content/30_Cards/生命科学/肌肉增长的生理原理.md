# 肌肉增长的生理原理

## 1. 肌肉结构基础

### 1.1 肌肉层级组织

| 层级 | 组成 | 功能 |
|---|---|---|
| 肌纤维束 (Fascicle) | 多根肌纤维 | 力的传递 |
| 肌纤维 (Muscle Fiber) | 肌原纤维组成 | 收缩单位 |
| 肌原纤维 (Myofibril) | 肌丝（粗/细） | 收缩基本单位 |
| 肌丝 | 肌球蛋白+肌动蛋白 | 产生力量 |

### 1.2 肌纤维类型

| 类型 | 收缩速度 | 抗疲劳性 | 供能方式 | 直径 |
|---|---|---|---|---|
| I 型（慢肌/红肌） | 慢 | 高 | 氧化磷酸化 | 较小 |
| IIa 型（中间型） | 中等 | 中等 | 混合供能 | 中等 |
| IIb/IIx 型（快肌/白肌） | 快 | 低 | 糖酵解 | 较大 |

## 2. 肌肉收缩的分子机制

### 2.1 滑动丝理论 (Sliding Filament Theory)

- **粗丝（肌球蛋白）**：形成横桥与细丝结合
- **细丝（肌动蛋白）**：在粗丝上滑动
- **收缩过程**：
  1. 神经冲动到达，Ca²⁺ 释放
  2. Ca²⁺ 与肌钙蛋白结合
  3. 原肌球蛋白构象改变，暴露结合位点
  4. 肌球蛋白横桥attach并水解ATP产生power stroke
  5. 细丝向M线方向滑动

### 2.2 兴奋-收缩耦联

```
动作电位 → T管膜去极化 → L-type Ca²⁺通道 → RyR受体激活 → 肌浆网Ca²⁺释放 → 收缩
```

## 3. 肌肉增长的机制 (Hypertrophy)

### 3.1 机械刺激与力学信号

| 信号通路 | 触发因素 | 效应 |
|---|---|---|
| mTORC1 | 机械牵张、氨基酸 | 促进蛋白质合成 |
| MAPK/ERK | 机械应力 | 细胞增殖 |
| PI3K/Akt | 生长因子 | 蛋白合成、抑制分解 |
| FAK | 张力刺激 | 整合素信号 |

### 3.2 代谢应激与体积效应

- **代谢产物积累**：H⁺、无机磷酸、AMP → 激活增殖信号
- **细胞肿胀**：渗透压改变可能触发合成代谢
- **激素环境**：
  - **睾酮**：促进卫星细胞活化
  - **生长激素 (GH)**：通过IGF-1介导促合成
  - **胰岛素**：增强氨基酸摄取
  - **皮质醇**：高浓度时抑制合成、促进分解

### 3.3 卫星细胞与肌核增加

- **卫星细胞 (Satellite Cells)**：位于肌膜与基底膜之间
- **激活流程**：
  1. 机械刺激 → 卫星细胞活化
  2. 增殖 → 成肌细胞
  3. 分化 → 融合进入肌纤维
  4. **肌核增加**：增加蛋白质合成能力
- **肌肉肥大 vs 增生**：
  - 肥大：现有纤维直径增大
  - 增生：纤维数量增加（人类中贡献较小）

### 3.4 蛋白质代谢平衡

```
蛋白质合成 (MPS) > 蛋白质分解 (MPB) → 净蛋白质正平衡 → 肌肉增长
```

| 影响因素 | 蛋白质合成 | 蛋白质分解 |
|---|---|---|
| 阻力训练 | ↑↑ | ↑ |
| 充足氨基酸 | ↑↑ | - |
| 胰岛素 | ↑ | ↓ |
| 皮质醇 | ↓ | ↑↑ |
| 力学刺激 | ↑↑ | - |

## 4. 肌肉增长的分期与恢复

### 4.1 训练后分期

| 时相 | 时间窗口 | 生理变化 |
|---|---|---|
| 急性期 | 0-4h | 肌肉损伤、炎症反应、Ca²⁺释放 |
| 合成代谢期 | 4-48h | 蛋白质合成峰值、卫星细胞激活 |
| 适应期 | 48-72h+ | 超量恢复、力量适应 |

### 4.2 超量恢复 (Supercompensation)

```
训练 → 消耗 → 恢复至基线 → 超量恢复 → 适应
         ↑_________|_____________|
                   时间
```

- 最佳训练时机：**超量恢复峰值**
- 过度训练：恢复不足导致分解 > 合成

## 5. 营养对肌肉增长的影响

### 5.1 蛋白质摄入

| 指标 | 推荐值 | 说明 |
|---|---|---|
| 每餐最低量 | 20-40g 完全蛋白 | 触发 MPS 最大应答 |
| 每日总量 | 1.6-2.2 g/kg体重 | 力量训练者 |
| Leucine阈值 | ~2.5-3g/餐 | 触发 MPS 关键信号 |
| 分布模式 | 每3-4h分次摄入 | 维持 MPS 峰值 |

### 5.2 碳水化合物的作用

- 补充糖原储备
- 胰岛素促氨基酸进入细胞
- 节约蛋白质（抗糖异生）

### 5.3 膳食脂肪

- 维持睾酮水平（适量脂肪摄入重要）
- 过低脂肪 → 睾酮下降

## 6. 合成代谢抵抗 (Anabolic Resistance)

### 6.1 影响因素

| 因素 | 影响机制 |
|---|---|
| 年龄增长 | mTOR敏感性下降、卫星细胞减少 |
| 久坐 | 肌肉代谢活性降低 |
| 肥胖 | 慢性炎症、胰岛素抵抗 |
| 营养不良 | 氨基酸可用性降低 |

### 6.2 应对策略

- 提高蛋白质摄入量（老年人需更多亮氨酸）
- 抗阻训练（最重要干预手段）
- 充足睡眠（生长激素分泌）
- 减少慢性炎症

## 7. 核心要点速记

1. 肌肉增长本质：**蛋白质合成 > 蛋白质分解**的净正平衡
2. 机械刺激通过 **mTORC1** 通路触发蛋白质合成
3. **卫星细胞**活化是肌肉生长的重要细胞来源
4. 阻力训练后 **超量恢复** 是训练适应的基础
5. **亮氨酸**是触发肌肉蛋白合成的关键氨基酸
6. 合成代谢抵抗随年龄增长而加重，需要更大刺激和蛋白质摄入
7. 休息和营养是实现肌肉增长的必要条件
