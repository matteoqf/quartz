# 食物过敏原分类

## 1. 基本概念

### 1.1 食物过敏

- 食物过敏是免疫系统对特定食物蛋白质的异常反应
- 当过敏原进入体内后，免疫系统将其识别为有害物质
- 产生 IgE 抗体介导的 I 型超敏反应（Immediate Hypersensitivity）

### 1.2 食物不耐受 vs 食物过敏

| 特征 | 食物过敏 | 食物不耐受 |
|---|---|---|
| 免疫参与 | IgE 介导 | 非免疫介导 |
| 反应时间 | 接触后数分钟至数小时 | 数小时至数天 |
| 剂量相关性 | 低剂量即可触发 | 通常与剂量相关 |
| 症状 | 荨麻疹、呼吸困难、休克等 | 腹胀、腹泻、头痛等 |

## 2. 过敏原分类系统

### 2.1 国际常见的过敏原分类

根据 Codex Alimentarius Commission（国际食品法典委员会）和各国法规，常见过敏原分为以下几类：

#### 8 大类常见过敏原

| 类别 | 代表食物 | 主要过敏蛋白 |
|---|---|---|
| **含麸质谷物** | 小麦、大麦、黑麦、燕麦 | 醇溶蛋白（gliadin）、麦谷蛋白（glutenin） |
| **甲壳类** | 虾、蟹、龙虾、磷虾 | 原肌球蛋白（tropomyosin）、精氨酸激酶 |
| **鱼类** | 鲑鱼、金枪鱼、鳕鱼、鲈鱼 | 鱼肌钙蛋白（parvalbumin）、鱼明胶 |
| **蛋类** | 鸡蛋、鸭蛋、鹌鹑蛋 | 卵清蛋白（ovalbumin）、卵类黏蛋白（ovomucoid） |
| **花生** | 花生及相关制品 | Ara h 1、Ara h 2、Ara h 3 |
| **大豆** | 黄豆、豆腐、豆浆、酱油 | Gly m 4、Gly m 5、Gly m 6 |
| **乳类** | 牛奶、羊奶、奶酪、酸奶 | 酪蛋白（casein）、乳清蛋白（whey） |
| **坚果类** | 杏仁、核桃、腰果、榛子、开心果 | 多种储存蛋白（2S albumins、legumins） |

#### 扩展过敏原（14 种）

除上述 8 大类外，以下物质也被多数国家列为标注要求：

| 类别 | 代表食物 |
|---|---|
| 芝麻 | 芝麻籽、芝麻油 |
| 软体动物 | 扇贝、鲍鱼、蛤蜊、章鱼 |
| 芥末 | 芥末籽、芥末酱 |
| 芹菜 | 芹菜茎、芹菜根 |
| 羽扇豆 | 羽扇豆粉、羽扇豆蛋白 |
| 燕麦 | 含麸质的燕麦制品 |
| 明胶 | 动物来源明胶 |
| 豌豆 | 豌豆、豌豆蛋白 |
| 羽扇豆 | 羽扇豆粉 |

### 2.2 按过敏原蛋白家族分类

| 蛋白家族 | 代表过敏原 | 特点 |
|---|---|---|
| ** Profilins** | Bet v 2（桦树）、Hev b 8（橡胶） | 交叉反应性强 |
| **PR-10 蛋白** | Bet v 1（桦树） | 与食物过敏原交叉反应 |
| **Cupin 超家族** | Ara h 1、Ara h 3（花生） | 热稳定性强 |
| **非特异性脂质转运蛋白（ns-LTP）** | Pru p 3（桃） | 广泛分布，热稳定性 |
| **肌钙蛋白（Parvalbumin）** | Gad c 1（鳕鱼） | 主要鱼过敏原 |

## 3. 过敏机制

### 3.1 I 型超敏反应机制

```
过敏原进入机体
      ↓
抗原呈递细胞（APC）处理并呈递过敏原肽
      ↓
Th2 细胞活化，分泌 IL-4、IL-5、IL-13
      ↓
B 细胞产生 IgE 抗体
      ↓
IgE 与肥大细胞/嗜碱性粒细胞表面的 FcεRI 受体结合
      ↓
再次接触同一过敏原
      ↓
过敏原交联 IgE-FcεRI 复合物
      ↓
肥大细胞脱颗粒，释放组胺、白三烯、前列腺素等介质
      ↓
过敏症状（血管扩张、平滑肌收缩、黏液分泌）
```

### 3.2 交叉反应

- **原理**：不同过敏原具有结构相似的抗原表位
- **常见交叉反应**：
  - 桦树花粉与苹果、樱桃、胡萝卜（PR-10 蛋白）
  - 乳胶与香蕉、鳄梨、猕猴桃（hevein 相关蛋白）
  - 虾与蟑螂、尘螨（原肌球蛋白）
  - 牛奶与牛肉（BSA 相关蛋白）

## 4. 临床分级

### 4.1 过敏反应严重程度分级

| 分级 | 症状 | 举例 |
|---|---|---|
| **轻度** | 局部荨麻疹、瘙痒、口唇麻木 | 口腔过敏综合征 |
| **中度** | 全身荨麻疹、血管性水肿、呼吸困难 | 全身性过敏反应 |
| **重度** | 过敏性休克、喉头水肿、呼吸衰竭 | 危及生命的紧急情况 |

### 4.2 主要症状

| 系统 | 症状 |
|---|---|
| **皮肤** | 荨麻疹、湿疹加重、血管性水肿、瘙痒 |
| **消化系统** | 口腔瘙痒、呕吐、腹泻、腹痛 |
| **呼吸系统** | 鼻塞、流涕、喉头水肿、哮喘加重 |
| **心血管系统** | 低血压、心悸、过敏性休克 |

## 5. 诊断与检测

### 5.1 检测方法

| 方法 | 原理 | 特点 |
|---|---|---|
| **皮肤点刺试验（SPT）** | 检测 IgE 介导的即时皮肤反应 | 快速、灵敏 |
| **血清特异性 IgE 检测** | 定量检测特定过敏原的 IgE | 可量化、不受药物影响 |
| **口服食物激发试验（OFC）** | 金标准诊断方法 | 可能诱发严重反应，需专业监护 |
| **斑贴试验** | 适用于迟发型过敏（IV 型） | 用于接触性皮炎 |

## 6. 管理与治疗

### 6.1 核心原则

- **严格回避**：完全避免已知过敏原
- **紧急处理**：随身携带肾上腺素自动注射器（严重过敏者）
- **营养替代**：确保营养均衡，避免食物种类单一

### 6.2 食物过敏原的替代方案

| 过敏原 | 替代食物 |
|---|---|
| 牛奶 | 植物奶（豆奶、杏仁奶、燕麦奶）、补钙食品 |
| 鸡蛋 | 亚麻籽+水混合物、豆腐、香蕉泥 |
| 小麦 | 无麸质面粉（米粉、荞麦、藜麦） |
| 花生 | 种子类（葵花籽、南瓜子）、坚果酱（注意交叉过敏） |
| 大豆 | 豌豆蛋白、扁豆 |

## 7. 核心要点速记

1. **8 大类常见过敏原**：含麸质谷物、甲壳类、鱼类、蛋类、花生、大豆、乳类、坚果类
2. 食物过敏由 **IgE 介导**，属 I 型超敏反应
3. 过敏原主要是**食物蛋白质**，不同食物间存在交叉反应
4. 诊断金标准是**口服食物激发试验（OFC）**
5. 管理的核心是**严格避免接触过敏原**
6. 严重过敏反应需使用**肾上腺素**急救
