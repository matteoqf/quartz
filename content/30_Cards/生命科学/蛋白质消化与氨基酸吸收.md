# 蛋白质消化与氨基酸吸收

## 1. 消化的基本过程

### 1.1 胃阶段

- **胃蛋白酶 (Pepsin)**：由胃主细胞分泌的胃蛋白酶原经胃酸激活转化而来
- **最适 pH**：约 1.5–2.5（胃酸提供酸性环境）
- **作用**：切割蛋白质内部的肽键，主要产物为多肽和少量游离氨基酸

### 1.2 小肠阶段（主要消化场所）

#### 胰腺分泌的酶

| 酶 | 前体 | 激活方式 | 作用位点 |
|---|---|---|---|
| 胰蛋白酶 (Trypsin) | 胰蛋白酶原 | 被肠激酶激活 | 切割 Lys、Arg 的羧基侧 |
| 胰凝乳酶 (Chymotrypsin) | 胰凝乳酶原 | 被胰蛋白酶激活 | 切割芳香族氨基酸侧链羧基 |
| 弹性蛋白酶 (Elastase) | 弹性蛋白酶原 | 被胰蛋白酶激活 | 切割小脂肪族氨基酸侧链 |
| 羧肽酶 (Carboxypeptidase) | 前体形式 | 被胰蛋白酶激活 | 从 C 端逐个切除氨基酸 |
| 氨肽酶 (Aminopeptidase) | 刷状缘膜酶 | — | 从 N 端逐个切除氨基酸 |

#### 刷状缘膜酶

- **寡肽酶 (Oligopeptidase)**：进一步水解短肽（二肽、三肽）

## 2. 吸收机制

### 2.1 吸收形式

- **游离氨基酸**：主要通过主动转运吸收
- **寡肽（2–3 肽）**：部分可直接被吸收，进入肠细胞后被胞内肽酶水解

### 2.2 转运系统

| 转运系统 | 底物 | 能量需求 | 备注 |
|---|---|---|---|
| Na⁺ 依赖性转运 | 中性氨基酸 | 主动转运（间接消耗 ATP） | 与 Na⁺ 共转运 |
| Na⁺ 依赖性转运 | 酸性氨基酸 | 主动转运 | 与 Na⁺ 共转运 |
| Na⁺ 依赖性转运 | 碱性氨基酸 | 主动转运 | 与 Na⁺ 共转运 |
| Na⁺ 依赖性转运 | 脯氨酸、羟脯氨酸 | 主动转运 | 特殊转运系统 |
| 非 Na⁺ 依赖性转运 | 某些中性氨基酸 | 易化扩散 | 系统 L |

### 2.3 吸收部位

- **小肠上皮细胞（空肠 > 回肠）**
- 氨基酸通过门静脉进入肝脏

## 3. 关键原理

### 3.1 蛋白质的互补效应

- 不同蛋白质的氨基酸组成不同，混合食用可弥补单一蛋白质的必需氨基酸缺陷
- 优质蛋白（完全蛋白）含全部必需氨基酸：动物性蛋白、乳清蛋白、鸡蛋蛋白

### 3.2 氮平衡

- **正氮平衡**：摄入 > 排出（生长、妊娠、恢复期）
- **负氮平衡**：摄入 < 排出（饥饿、疾病、应激）
- **零氮平衡**：摄入 = 排出（维持阶段）

### 3.3 肠道屏障与过敏

- 未完全消化的蛋白质片段（大分子肽）可能引发食物过敏
- 肠道屏障功能受损时，过敏风险增加

## 4. 核心要点速记

1. 蛋白质消化的主要场所是**小肠**
2. 胰蛋白酶是**关键激活酶**，可激活其他胰酶原
3. 氨基酸吸收主要依赖**Na⁺ 依赖性主动转运**
4. 二肽、三肽可被**完整吸收**后胞内水解
5. 必需氨基酸必须从食物中获取，体内不能合成
