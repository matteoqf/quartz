# 体温调节发热与中暑

## 核心概念

体温恒定是机体正常生理活动的基础。正常人核心体温约为 **36.5–37.5°C**，通过产热与散热的动态平衡维持稳态。发热（感染性/炎症性）与中暑（热射病）是两种性质不同的体温升高，其机制、处理原则及预后均有显著差异。

---

## 1. 正常体温调节机制

### 体温平衡方程

```
产热 = 散热
```

### 产热途径
- **基础代谢**：肝脏、脑、心脏产热占基础代谢的60%
- **骨骼肌收缩**：寒战产热（非寒战产热通过棕色脂肪组织）
- **食物特殊动力效应**：消化吸收过程中的产热
- **交感神经兴奋**：促进去甲肾上腺素释放，增加脂肪组织产热

### 散热途径

| 散热方式 | 占比 | 条件 |
|----------|------|------|
| 辐射散热 | ~60% | 环境温度低于体表温度 |
| 传导散热 | ~15% | 与冷物体接触 |
| 对流散热 | ~15% | 空气流动带走体表热量 |
| 蒸发散热 | ~10% | 出汗时；高温环境下可大幅增加 |

### 体温调定点（Set Point）
下丘脑视前区存在**体温调定点**，正常值约37°C。通过负反馈机制：
- 体温 > 调定点 → 散热增加（血管扩张、出汗）
- 体温 < 调定点 → 产热增加（血管收缩、寒战）

---

## 2. 发热（Fever）

### 定义
发热是由于**内源性致热原（IL-1、IL-6、TNF-α、PGE₂）**作用于下丘脑，使体温调定点**升高**而出现的体温升高。

### 发热与过热的区别

| 特征 | 发热 | 过热（高热） |
|------|------|-------------|
| 调定点 | 升高 | 正常 |
| 机制 | 致热原 → 下丘脑调定点上移 | 散热障碍或产热过度 |
| 主动/被动 | 主动调节 | 被动失调 |
| 可控性 | 可被退热药抑制 | 退热药效果有限 |

### 发热分期

1. **体温上升期**：调定点上移 → 畏寒、寒战、皮肤血管收缩
2. **高温持续期**：体温达到新调定点 → 自觉酷热、面红、心率加快
3. **体温下降期**：致热原消退 → 调定点恢复 → 出汗散热

### 常见病因
- **感染性**：细菌（革兰氏阴性菌内毒素）、病毒、真菌
- **非感染性**：无菌性炎症（手术、创伤）、肿瘤（尤其淋巴瘤）、自身免疫病、药物热

### 处理原则
- **病因治疗**：抗感染、消除炎症
- **对症处理**：
  - 体温 > 38.5°C 或有不适感 → 物理降温（温水擦浴、冰敷大动脉）
  - 药物：扑热息痛（对乙酰氨基酚）、布洛芬（抑制COX，减少PGE₂合成）
- **监测**：关注体温曲线、伴随症状、脏器功能

## 3. 中暑（Heat-Related Illness）

### 病因
暴露于高温环境 + **散热障碍**或**产热增加**，超出机体代偿能力。

### 病情进展谱

```
先兆中暑 → 轻症中暑 → 重症中暑（热痉挛→热衰竭→热射病）
```

| 类型 | 临床表现 | 处理 |
|------|----------|------|
| **先兆中暑** | 头晕、乏力、多汗、口渴 | 转移至阴凉处，补水 |
| **轻症中暑** | 面色潮红、脉搏快、恶心 | 降温 + 补液 |
| **热痉挛** | 肌肉痛性痉挛（大汗后） | 补钠，降温 |
| **热衰竭** | 头晕、恶心、呕吐、晕厥（血压低） | 快速补液，纠正休克 |
| **热射病** | 高热（>40°C）+ 意识障碍 + 多器官功能障碍 | **紧急降温 + 生命支持** |

### 热射病（Heat Stroke）

**是最严重的中暑类型，死亡率高达 20–70%。**

#### 分类
- **经典型（非劳力性）**：多见于老人、幼儿、慢性病患者，高温环境下散热障碍
- **劳力型**：多见于年轻人，高温 + 高湿 + 剧烈运动，产热急剧增加

#### 病理生理
- 核心体温 > 40°C → 蛋白质变性、酶系统紊乱
- 全身炎症反应综合征（SIRS）→ 毛细血管内皮损伤 → 凝血功能障碍
- 脏器灌注不足 → 多器官功能障碍（ARDS、DIC、急性肾损伤、肝损伤）

#### 诊断标准
- 核心体温 ≥ 40°C
- 中枢神经系统功能障碍（谵妄、昏迷、抽搐）
- 排除其他原因

#### 紧急处理

**"降温—补液—转送"三步走：**

1. **快速降温**（首要措施，目标：30 min内将体温降至 < 39°C）
   - 物理降温：冰水浴（最佳）、蒸发降温（凉水喷洒 + 风扇）、冰敷颈动脉/腋窝/腹股沟
   - 药物降温：**无效**，且可能加重肝损伤，不推荐

2. **补液扩容**
   - 轻度无需特殊补液
   - 热衰竭需快速静脉输注生理盐水

3. **生命支持**
   - 保持气道通畅
   - 纠正凝血功能障碍
   - 防治急性肾损伤、ARDS
   - 监测心肝肾等脏器功能

## 4. 发热与中暑的鉴别要点

| 鉴别点 | 发热 | 中暑 |
|--------|------|------|
| 病因 | 感染/炎症致热原 | 高温环境暴露 |
| 调定点 | 升高 | 正常 |
| 出汗 | 常有（退热期） | 可有或无（热射病常无汗） |
| 皮肤 | 多干燥或潮湿 | 多干燥、灼热（热射病） |
| 伴随症状 | 原发病症状 | 意识障碍、抽搐 |
| 处置优先级 | 病因+对症 | **快速降温** |
| 退热药效果 | 有效 | 无效/有害 |

---

## 5. 临床病例要点总结

- **发热是生理性防御反应**，适度发热（< 38.5°C）有利于免疫细胞活性增强，不必过度积极退热
- **中暑的本质是散热失败**，降温速度直接决定预后，"黄金30分钟"内降温至 39°C 以下可显著降低死亡率
- **热射病不是严重发热**，不能用发热的思维处理，退热药无效且可能加重病情
- 高危人群（老人、儿童、慢性病患者）在高温天气需**提前预防**：充足补水、避免正午外出、保持室内通风

---

## 总结

发热与中暑虽然都表现为体温升高，但机制截然不同。发热是机体对感染和炎症的主动防御反应，调定点上移，退热药有效；中暑是被动过热过程，调定点正常，核心体温管理（而非药物）是治疗核心。临床遇到高热患者，需仔细询问病史（暴露史）、查体（皮肤湿度、意识状态），准确鉴别，才能采取正确的救治措施。
