# 维生素D合成与钙吸收

## 1. 维生素D的合成

### 1.1 皮肤合成（内源性）

- **原料**: 表皮中的7-脱氢胆固醇（7-Dehydrocholesterol）
- **触发**: 紫外线B（UVB，波长290-315 nm）照射
- **场所**: 表皮基底层的角质形成细胞
- **产物**: 维生素D₃（前体 cholecalciferol）
- **注意**: 过度防晒会显著减少合成；肤色深、衣物遮盖、冬季高纬度地区均会削弱

### 1.2 食物来源（外源性）

| 来源 | 维生素D形式 |
|------|------------|
| 深海鱼肝油 | D₃ |
| 富含脂肪的鱼类（三文鱼、鲭鱼） | D₃ |
| 蛋黄 | D₃ |
| 强化食品（牛奶、谷物） | D₂ 或 D₃ |
| 植物来源（蘑菇、酵母） | D₂（麦角钙化醇）|

---

## 2. 维生素D的活化（两步羟基化）

### 第一步：肝脏活化

- **酶**: 25-羟化酶（25-hydroxylase）
- **产物**: 25-羟胆钙化醇（25(OH)D，骨化二醇）
- **特点**: 体内储存形式，半衰期约2-3周；临床检测指标

### 第二步：肾脏活化

- **酶**: 1α-羟化酶（1α-hydroxylase）
- **产物**: 1,25-二羟胆钙化醇（1,25(OH)₂D，骨化三醇）
- **活性形式**: 与维生素D受体（VDR）结合发挥作用
- **调控**: 受PTH、血磷、FGF23反馈调节

---

## 3. 维生素D的生理功能

### 3.1 促进肠道钙吸收（核心功能）

- **机制**: 诱导小肠上皮细胞合成钙通道蛋白（TRPV6）和钙结合蛋白（Calbindin）
- **效果**: 主动转运吸收钙，效率可提升3-4倍
- **过程**: 1,25(OH)₂D → 进入肠细胞 → 与VDR/RXR结合 → 激活靶基因转录 → 增加TRPV6、Calbindin表达

### 3.2 其他功能

- 促进肾小管对钙的重吸收
- 调节骨矿化（双向调控：促进成骨、动员骨钙）
- 调节血钙血磷平衡
- 免疫调节、抗炎作用
- 细胞增殖与分化调控

---

## 4. 钙的吸收机制

### 4.1 吸收形式

| 方式 | 占比 | 特点 |
|------|------|------|
| 主动转运（跨细胞途径） | ~45% | 1,25(OH)₂D依赖，十二指肠为主 |
| 被动扩散（旁细胞途径） | ~55% | 浓度梯度驱动，回肠为主 |

### 4.2 主动转运步骤

1. 肠腔钙离子 → TRPV6通道进入肠细胞
2. 细胞内钙结合蛋白（Calbindin-D9k）结合并转运钙
3. Ca²⁺-ATP酶（PMCAl）泵出至血液侧
4. 进入血液循环，与白蛋白结合或以离子形式运输

---

## 5. 调控机制

### 5.1 正向调控（PTH轴）

```
低血钙 → 甲状旁腺分泌PTH → 肾脏1α-羟化酶↑ → 1,25(OH)₂D↑ → 钙吸收↑
```

### 5.2 负向调控（FGF23轴）

```
高血钙/高磷 → FGF23↑ → 抑制1α-羟化酶，激活24-羟化酶 → 1,25(OH)₂D↓ → 钙吸收↓
```

---

## 6. 临床关联

| 指标 | 说明 |
|------|------|
| 血清25(OH)D | 评估维生素D状态的金标准；>30 ng/mL为充足 |
| 1,25(OH)₂D | 活性形式，但因半衰期短（4-6小时），不用于营养评估 |
| VDR多态性 | 影响个体对维生素D的敏感性 |
| 缺乏后果 | 佝偻病（儿童）、骨软化症（成人）、骨质疏松 |

---

## 7. 实验/研究相关

### 常用检测方法

- **化学发光免疫法（CLIA）**: 检测25(OH)D
- **HPLC/LC-MS/MS**: 精确分型D₂/D₃的25羟化代谢物
- **放射受体法（RRA）**: 测定1,25(OH)₂D活性

### 信号通路关键词

- VDR（Vitamin D Receptor）
- RXR（Retinoid X Receptor）
- TRPV6（Transient Receptor Potential Vanilloid 6）
- Calbindin-D9k
- CYP27B1（1α-羟化酶）
- CYP24A1（24-羟化酶，分解代谢）
