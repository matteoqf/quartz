# 抗生素作用原理与耐药性

## 核心概念

抗生素是由微生物（真菌、细菌）产生的化学物质，能抑制或杀死其他微生物。抗菌机制主要分为五类：**抑制细胞壁合成**、**抑制蛋白质合成**、**抑制核酸合成**、**破坏细胞膜完整性**、**抑制叶酸代谢**。耐药性是微生物通过基因突变或基因转移获得的对抗生素的抵抗能力，已成为全球公共卫生重大挑战。

---

## 1. 抗生素的五大作用机制

### 1.1 抑制细胞壁合成

细菌细胞壁是由肽聚糖（peptidoglycan）交联形成的高分子网状结构，维持细菌渗透压平衡。哺乳动物细胞无细胞壁，因此这类抗生素对人体毒性较低。

**代表性药物：**
- **β-内酰胺类**（青霉素、头孢菌素）：抑制转肽酶，阻止肽聚糖交联
- **万古霉素**：阻断肽聚糖前体的脂质载体循环
- **碳青霉烯类**（亚胺培南）：广谱β-内酰胺，抑制多种转肽酶

**作用机制：**
```
肽聚糖合成 → 转肽酶交联 → 刚性细胞壁
     ↑
β-内酰胺类竞争结合转肽酶（青霉素结合蛋白，PBP）
     ↓
交联中断 → 细胞壁强度下降 → 细菌裂解（渗透压冲击）
```

| 药物类型 | 作用靶点 | 抗菌谱 |
|---------|---------|--------|
| 青霉素G | PBP1, PBP3 | 革兰氏阳性菌 |
| 头孢曲松 | PBP2 | 广谱，革兰氏阳/阴性 |
| 万古霉素 | D-Ala-D-Ala | 革兰氏阳性菌（ MRSA） |

### 1.2 抑制蛋白质合成

细菌核糖体为70S（30S + 50S），与哺乳动物80S核糖体不同，使得抗生素具有选择性毒性。

**30S亚基靶点：**
- **氨基糖苷类**（庆大霉素、链霉素）：结合30S亚基，阻止起始复合物形成，导致错读
- **四环素类**：占据30S亚基的A位点，阻断氨酰-tRNA进入

**50S亚基靶点：**
- **大环内酯类**（红霉素、阿奇霉素）：结合50S亚基，阻断肽酰转移
- **氯霉素**：抑制肽酰转移酶活性
- **林可酰胺类**（克林霉素）：结合50S亚基，抑制肽链延伸
- **恶唑烷酮类**（利奈唑胺）：结合50S亚基，阻止起始复合物

```
70S核糖体
├── 30S亚基
│   ├── 16S rRNA（氨基糖苷、四环素靶点）
│   └── 蛋白质（S12等）
└── 50S亚基
    ├── 23S rRNA（大环内酯、氯霉素、林可酰胺靶点）
    └── 蛋白质（L27等）
```

| 抗生素 | 亚基 | 作用阶段 |
|-------|-----|---------|
| 庆大霉素 | 30S | 起始复合物+错读 |
| 四环素 | 30S | 氨酰-tRNA进入 |
| 红霉素 | 50S | 肽链延伸 |
| 氯霉素 | 50S | 肽酰转移 |

### 1.3 抑制核酸合成

#### 1.3.1 抑制DNA复制

- **喹诺酮类**（环丙沙星、左氧氟沙星）：抑制DNA旋转酶（拓扑异构酶II）和拓扑异构酶IV，阻止DNA超螺旋解开，导致复制障碍

```
DNA复制：解旋 → 拓扑异构酶松开超螺旋 → 复制叉前进
              ↑
        喹诺酮类抑制拓扑异构酶
              ↓
        DNA双链断裂 → 细菌死亡
```

- **利福平**：抑制DNA依赖的RNA聚合酶，阻断转录

#### 1.3.2 抑制RNA合成

- **利福平/利福平类**：结合RNA聚合酶β亚基，阻止mRNA合成

| 药物 | 靶点 | 结果 |
|-----|-----|-----|
| 环丙沙星 | DNA旋转酶 | DNA复制受阻 |
| 利福平 | RNA聚合酶 | 转录中断 |

### 1.4 破坏细胞膜完整性

细菌细胞膜为磷脂双分子层，功能类似哺乳动物细胞膜。多黏菌素类可与脂多糖（LPS）结合，破坏外膜完整性。

**代表性药物：**
- **多黏菌素B/E**：结合革兰氏阴性菌外膜的脂多糖，破坏膜屏障
- **达托霉素**：插入革兰氏阳性菌细胞膜，形成离子通道

### 1.5 抑制叶酸代谢

叶酸是DNA合成必需的辅酶。磺胺类药物竞争性抑制二氢叶酸合成酶，阻断二氢叶酸合成。

```
叶酸合成途径：
PABA → 二氢叶酸合成酶 → 二氢叶酸 → 四氢叶酸（辅酶形式）
          ↑
    磺胺类竞争抑制
```

**甲氧苄啶（TMP）**：抑制二氢叶酸还原酶，与磺胺类协同作用（SMZ-TMP复方）

## 2. 耐药性机制

### 2.1 耐药性产生的分子机制

#### 2.1.1 酶介导的耐药性

**β-内酰胺酶：**
- 革兰氏阳性菌（葡萄球菌）产生青霉素酶
- 革兰氏阴性菌产生超广谱β-内酰胺酶（ESBL）
- AmpC β-内酰胺酶（头孢菌素酶）
- 碳青霉烯酶（KPC、NDM、OXA-48）

```
青霉素 → [β-内酰胺酶] → 无活性青霉素酸
          ↑
     基因突变或质粒获得
```

**氨基糖苷修饰酶：** 乙酰转移酶、磷酸转移酶、核苷转移酶使药物失活

**大环内酯钝化酶：** 红霉素酯酶、水解酶

#### 2.1.2 靶点改变

- **PBP改变**： MRSA的PBP2a替代正常PBP，对β-内酰胺类亲和力极低
- **DNA旋转酶突变**：喹诺酮类耐药决定区（QRDR）突变
- **核糖体靶点突变**：甲基化修饰（erm基因编码23S rRNA甲基转移酶）

#### 2.1.3 外排泵（Efflux Pumps）

主动将抗生素泵出菌体，降低胞内药物浓度。

| 外排泵类型 | 底物 | 代表菌种 |
|-----------|-----|---------|
| AdeABC | 氨基糖苷、四环素 | 鲍曼不动杆菌 |
| MexAB-OprM | 氟喹诺酮、β-内酰胺 | 铜绿假单胞菌 |
| MefA | 大环内酯 | 链球菌 |

#### 2.1.4 渗透性改变

- **外膜孔蛋白丢失或减少**：革兰氏阴性菌下调OmpF、OmpC等孔蛋白
- **生物膜形成**：胞外聚合物基质（EPS）阻碍抗生素渗透

### 2.2 耐药基因的传播

#### 2.2.1 垂直传播
染色体突变导致耐药性状遗传，子代继承耐药基因。

#### 2.2.2 水平基因转移（HGT）

```
        转化（Transformation）
            ↑
质粒接合 ←—— 细菌
            ↓
        转导（Transduction）
```

- **接合**：F质粒/R质粒通过性菌毛在细菌间转移耐药基因（最常见）
- **转导**：噬菌体介导的基因转移
- **转化**：直接摄取环境中的裸露DNA

**整合子-基因盒系统：** 捕获和表达耐药基因，驱动多重耐药传播

### 2.3 主要耐药菌与机制对应

| 耐药菌 | 简称 | 主要耐药机制 |
|-------|-----|------------|
| 甲氧西林耐药金黄色葡萄球菌 | MRSA | PBP2a（靶点改变） |
| 万古霉素耐药肠球菌 | VRE | 肽聚糖前体改变（D-Ala→D-Lac） |
| 超广谱β-内酰胺酶肠杆菌 | ESBL+ | β-内酰胺酶水解 |
| 碳青霉烯耐药肠杆菌 | CRE | KPC/NDM碳青霉烯酶 |
| 耐多药结核分枝杆菌 | MDR-TB | 细胞壁通透性+外排泵 |
| 泛耐药铜绿假单胞菌 | XDR-PA | 多重机制（外膜屏障+外排泵+酶） |

## 3. 耐药性演化动力学

### 3.1 选择压力与"抗菌药物干预"

抗生素使用创造选择压力，敏感菌被抑制或杀死，耐药菌获得生长优势：

```
高抗生素浓度 → 敏感菌↓ + 耐药菌优势生长 → 耐药流行
中断/低浓度抗生素 → 敏感菌竞争恢复 → 耐药比例下降（理论上）
```

### 3.2 最小抑菌浓度（MIC）与临床折点

| 参数 | 定义 |
|-----|-----|
| MIC | 抑制可见生长的最低药物浓度 |
| MBC | 杀死99.9%起始菌量的最低浓度 |
| 敏感（S） | MIC ≤ 折点，常规剂量有效 |
| 耐药（R） | MIC > 折点，常规剂量无效 |

### 3.3 耐药性防控原则

- **合理用药**：根据药敏试验选药，避免滥用
- **联合用药**：减少耐药突变概率（尤其抗结核治疗）
- **轮换用药**：策略性换药恢复旧药敏感性
- **感染控制**：隔离耐药菌患者，切断传播链
- **新药研发**：新型β-内酰胺酶抑制剂组合（如阿维巴坦）
