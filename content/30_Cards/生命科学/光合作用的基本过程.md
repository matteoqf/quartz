# 光合作用的基本过程

## 核心概念

光合作用是光合生物（植物、藻类、蓝细菌）利用光能将 CO₂ 和 H₂O 合成有机物（葡萄糖）并释放 O₂ 的过程。总反应式：

```
6 CO₂ + 6 H₂O + 光能 → C₆H₁₂O₆ + 6 O₂
```

光合作用分为**光反应**（类囊体膜）和**暗反应**（叶绿体基质）两个阶段。**光反应**将光能转化为化学能（ATP + NADPH），**暗反应**（Calvin循环）利用这些能量将 CO₂ 固定为糖。

---

## 1. 光合作用的细胞器：叶绿体

| 区域 | 位置 | 功能 |
|------|------|------|
| 外膜 | 外层膜 | 物质进出 |
| 内膜 | 内层膜 | 物质进出 |
| 基质 | 叶绿体内部 | 暗反应（Calvin循环）进行 |
| 类囊体 | 堆叠成基粒 | 光反应进行 |
| 类囊体膜 | 膜结构 | 包含光系统II、光系统I、细胞色素b₆f复合体、ATP合酶 |

---

## 2. 光反应（Light Reactions）

### 概述
光反应发生在类囊体膜上，通过**光系统II（PSII）**和**光系统I（PSI）**两个光系统的协同作用，将光能转化为 ATP 和 NADPH，并释放 O₂。

### 关键步骤

#### （1）水的光解
```
2 H₂O → 4 H⁺ + 4 e⁻ + O₂
```
- 由 PSII 的氧化引发电位（P680*）驱动
- 释放 O₂，是大气 O₂ 的唯一来源
- 4 个电子传递给 PSII 反应中心

#### （2）电子传递链（Z scheme）
```
PSII → 质体醌（PQ）→ 细胞色素b₆f复合体 → 质体蓝素（PC）→ PSI → 质体铁氧还蛋白（Fd）
```

| 步骤 | 事件 | 产岆 |
|------|------|------|
| PSII光驱动 | P680被激发为P680*，获得高能电子 | 电子进入电子传递链 |
| 水的光解 | 补充PSII电子，释放O₂和H⁺ | H⁺积累于类囊体腔 |
| PQ循环 | 传递电子并泵送H⁺至类囊体腔 | 建立质子梯度 |
| 细胞色素b₆f | 传递电子，泵送H⁺ | 继续建立质子梯度 |
| PSI光驱动 | P700被激发，将电子传给Fd | NADP⁺还原为NADPH |

#### （3）化学渗透假说（光合磷酸化）
- 类囊体腔内 H⁺ 积累形成电化学梯度
- H⁺ 通过 **ATP 合酶** 流回基质，驱动 ADP + Pi → ATP
- 每产生 1 分子 O₂（约每 4 个光子），可合成约 **3 ATP**

### 光反应总览
```
2 H₂O + 2 NADP⁺ + 3 ADP + 3 Pi + 光能 
    → O₂ + 2 NADPH + 3 ATP + 3 H₂O
```

## 3. Calvin循环（暗反应 / 碳固定反应）

### 概述
Calvin循环在叶绿体基质中进行，利用光反应产生的 ATP 和 NADPH，将 CO₂ 固定并还原为甘油醛-3-磷酸（G3P），最终合成葡萄糖或其他有机物。**不需要光**（但并非只能发生在暗处）。

### 三个阶段

#### 阶段1：碳固定
| 反应 | 酶 |
|------|-----|
| CO₂ + 1,5-二磷酸核酮糖（RuBP）→ 2 甘油酸-3-磷酸（3-PGA）| **核酮糖-1,5-二磷酸羧化酶（Rubisco）** |

Rubisco 是地球上最丰富的酶，是碳固定的关键限速酶。

#### 阶段2：还原阶段
| 反应 | 酶 |
|------|-----|
| 3-PGA + ATP → 1,3-二磷酸甘油酸 + ADP | 磷酸甘油酸激酶 |
| 1,3-二磷酸甘油酸 + NADPH → 甘油醛-3-磷酸（G3P） + NADP⁺ + Pi | 甘油醛-3-磷酸脱氢酶 |

#### 阶段3：RuBP 再生
6 个 G3P 经过一系列反应（涉及多种酶），其中 1 个 G3P 离开循环用于糖合成，其余 5 个再生为 3 个 RuBP。

### Calvin 循环总览
```
3 CO₂ + 9 ATP + 6 NADPH + 5 H₂O 
    → 甘油醛-3-磷酸（G3P） + 9 ADP + 8 Pi + 6 NADP⁺ + 3 H⁺
```

要合成 1 分子葡萄糖，需要 **6 次循环**（固定 6 CO₂），净消耗 **18 ATP + 12 NADPH**。

## 4. 光呼吸（Photorespiration）

### 概念
当 O₂ 浓度高而 CO₂ 浓度低时，Rubisco 有机会催化 O₂ 与 RuBP 的反应，生成 3-PGA 和磷酸乙醇酸（2-碳化合物），这一过程称为**光呼吸**。

```
RuBP + O₂ → 3-PGA + 磷酸乙醇酸
```

### 问题
- 磷酸乙醇酸被代谢后释放 CO₂，但不产生 ATP 或 NADPH
- 相当于"浪费"了原本用于碳固定的 RuBP 和能量
- 在高温、干燥、高光强条件下尤为严重（此时气孔关闭，CO₂ 供应不足）

### C₃ 与 C₄ 植物
| 类型 | 代表植物 | 特点 |
|------|----------|------|
| C₃ 植物 | 水稻、小麦、大豆 | 直接进行 Calvin 循环，光呼吸严重 |
| C₄ 植物 | 玉米、甘蔗、高粱 | 先将 CO₂ 浓缩至 C₄ 化合物，再释放给 Calvin 循环，光呼吸极低 |

---

## 5. 光反应与 Calvin 循环的关系

```
光能
  ↓
类囊体膜（光反应）
  ↓
ATP + NADPH + O₂
  ↓
叶绿体基质（Calvin 循环）
  ↓
甘油醛-3-磷酸（G3P）
  ↓
葡萄糖及其他有机物
```

## 6. 光合作用与呼吸作用的对比

| 特征 | 光合作用 | 细胞呼吸 |
|------|----------|----------|
| 场所 | 叶绿体 | 细胞质 + 线粒体 |
| 能量转化 | 光能 → 化学能 | 化学能 → ATP |
| 原料 | CO₂ + H₂O | 有机物 + O₂ |
| 产物 | 有机物 + O₂ | CO₂ + H₂O + ATP |
| 方程逆关系 | 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂ | C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O |
| 能量流 | 能量储存（吸能） | 能量释放（放能）|
| 气氛 | 白天进行 | 持续进行 |

## 7. 影响光合作用的因素

| 因素 | 效应 |
|------|------|
| **光照强度** | 光补偿点以上时，光强↑则光合速率↑；光饱和点后不再增加 |
| **CO₂ 浓度** | CO₂ 浓度↑则光合速率↑（温室增产原理）|
| **温度** | 最适温度（约25–30°C）时速率最高；过高导致酶失活 |
| **水分** | 缺水导致气孔关闭，减少 CO₂ 吸收；过度蒸腾 |
| **矿质元素** | 氮、叶绿素合成相关；镁是叶绿素中心原子 |
| **光质** | 叶绿素主要吸收红光和蓝光；绿光反射（所以植物呈绿色）|

---

## 8. 临床/实际关联

- **叶绿素缺乏**：氮肥不足导致叶色发黄，光合作用减弱
- **光合作用与农业**：间作、套种利用不同光合效率；C₄作物在高光强高温地区产量更高
- **光合细菌**：蓝细菌可进行光合作用，是早期地球上产生氧气的主要来源
- **光抑制**：过强光照破坏 PSII 反应中心（D1蛋白损伤），植物通过非光化学猝灭保护

---

## 总结

光合作用通过**光反应**和**Calvin循环**两个阶段，将光能转化为储存在有机物中的化学能。**光反应**在类囊体膜上进行，通过光系统II和I及电子传递链产生 ATP 和 NADPH，并释放 O₂；**Calvin循环**在基质中进行，利用这些能量将 CO₂ 固定为糖。光呼吸是 Rubisco 在高氧低 CO₂ 条件下的副反应，会降低光合效率。C₄植物通过空间上的 CO₂ 浓缩机制有效抑制光呼吸。光合作用与细胞呼吸构成碳循环的核心互补过程。
