# 🔥 灾害防御与逃生

## 高层火灾：逃生 vs 固守

### 决策树

```
是否在着火层以上？
  ├── 是 → 立即逃生（向下跑）
  │         ├── 楼梯间逃生，不要乘电梯
  │         ├── 用湿毛巾捂口鼻（至少3层）
  │         └── 身体贴墙，弯腰前进
  │
  └── 否（在着火层/以下）
          ├── 如果楼梯间有烟/火 → 固守待援
          │   ├── 关好门，用湿毛巾堵门缝
          │   ├── 往门上浇水降低温度
          │   ├── 打开窗户（不跳！）
          │   └── 等待消防员（挥舞颜色鲜艳物品）
          │
          └── 如果楼梯间无烟/火 → 评估后再决定
```

### 固守待援的判断条件

```
符合以下条件 → 固守待援（比逃跑更安全）：
├── 楼梯间已经被烟/火封锁
├── 门外温度较低（用手背贴门判断）
├── 房间有窗户可以通风
└── 楼层较高（跳楼/冲火场必死）

不符合 → 立刻逃生
├── 门外已感觉烫手（烟火已到门口）
├── 窗户已被火焰封锁
└── 房间内已有烟（说明通风口被引入）
```

## 灭火器类型选择

| 类型 | 介质 | 绝缘性 | 适用场景 | 禁用 |
|------|------|--------|---------|------|
| 水基型 | 水+添加剂 | ❌ 导电 | 普通固体温 | 电器/油脂 |
| 干粉型 | 磷酸铵盐 | ✅ 绝缘 | 通用（ABC型） | 精密仪器 |
| CO₂型 | 二氧化碳 | ✅ 绝缘 | 电子设备/精密仪器 | 固体深部燃烧 |
| 泡沫型 | 泡沫 | ❌ 导电 | 油脂/液体 | 电器 |

> 家用推荐：**ABC干粉灭火器**（适用范围最广）或**水基型**（对人有安全掩护）

## 地震："伏地、遮挡、手抓牢"

### 正确做法

```
室内地震应对（"DCH"原则）：

Drop（伏地）：
  → 立即降低重心

Cover（遮挡）：
  ├── 坚固桌子下 → 桌下是最佳掩护
  ├── 没有桌子 → 靠内墙蹲下
  └── 远离窗户/玻璃/重物/灯具

Hold on（手抓牢）：
  → 抓住桌腿，防止桌子移动
  → 如果桌子移动，你跟着它移动，保持在桌下

室内避险禁区：
  ├── 门口（门框可能变形，塌下来比房间更危险）
  ├── 外墙（窗户/玻璃可能碎裂）
  ├── 吊灯/风扇/重物下方
  └── 电梯
```

### "黄金三角"是伪科学

```
❌ 地震"黄金三角"（躲在家具旁/建筑物倒塌会形成存活空间）：
  → 这个理论来自对倒塌建筑的误解
  → 地震中建筑物倒塌方向不可预测
  → 家具会被甩动/滑动，位置不稳定
  → 躲在家具旁可能被压死

✅ 国际通行的DCH原则（伏地遮挡手抓牢）：
  → 桌子下有明确坚固的遮挡
  → 跟着家具移动，保持在遮挡下
  → 得到全球救援机构认可
```
