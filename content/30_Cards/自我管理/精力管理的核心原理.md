# 精力管理的核心原理

## 精力金字塔

精力来源于四个相互关联的维度，呈金字塔结构：

1. **体能** — 最基础层，决定精力的上限
2. **情绪** — 影响精力的质量与稳定性
3. **思维** — 决定专注力与创造力
4. **意志** — 最顶层，赋予精力方向与意义

> 精力管理不是时间管理，而是管理你对自身状态的感知与调节。

## 核心原理

### 1. 精力是消耗与恢复的循环

- 精力不是线性资源，而是周期性可再生资源
- 高强度工作后必须主动休息，否则进入透支状态
- 恢复不是浪费时间，是为下一轮输出做准备

### 2. 遵循90分钟节律

- 人体的 ultradian rhythm（超昼夜节律）约90分钟一个周期
- 每90分钟安排短暂休息，维持精力在高位
- 强行跨越周期会导致效率骤降

### 3. 优先级决定精力分配

- 把最好的精力时段分配给最重要的事
- 低价值任务在精力低谷期处理
- 不要用高精力做低价值事务

### 4. 主动管理而非被动应对

- 提前规划精力分配，预判高/低谷
- 识别自己的精力模式，在高峰做创造，低谷做机械事务
- 压力本身不是问题，持续压力无恢复才是

### 5. 习惯比意志力更可靠

- 精力管理最终靠系统和习惯，而非临时调动意志力
- 建立固定的节奏和仪式感，减少决策消耗
- 睡眠、运动、饮食是精力的三大基石

## 实践原则

| 原则 | 说明 |
|------|------|
| 间歇性节奏 | 工作-休息交替，而非拉锯战 |
| 优先级第一 | 先做最重要的，而非最先想到的 |
| 尊重恢复 | 休息时真正断开，而非"边休息边工作" |
| 觉察状态 | 时刻监控自身能量水平，及时调整 |
| 长期视角 | 今天透支的精力，明天要还 |

---

*精力管理的本质：你无法管理时间，但你可以管理自己在时间中的状态。*
