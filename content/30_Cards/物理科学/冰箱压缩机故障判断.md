# 冰箱压缩机故障判断

## 1. 前期准备

- 断开冰箱电源，等待 5 分钟以上，使系统压力平衡
- 准备工具：万用表、兆欧表（摇表）、电流表、螺丝刀、钳子等

## 2. 外观与基础检查

- 观察压缩机外壳是否有明显变形、锈蚀、裂纹
- 检查压缩机接线端子是否松动、烧焦、腐蚀
- 听压缩机运行时是否有异常噪音（明显金属撞击声、摩擦声）
- 触摸压缩机外壳，正常运行时应有明显发热（50–80°C），过冷表明缺制冷剂，过热表明内部故障

## 3. 电气检测

### 3.1 绕组电阻检测

- 拔除压缩机接线，用万用表测量三相或单相绕组端子间电阻：
  - 三相压缩机：三组绕组两两之间电阻应相等（偏差 < 5%）
  - 单相压缩机：启动绕组与运行绕组之间电阻之和 ≈ 运行绕组与公共端电阻 + 启动绕组与公共端电阻
  - 电阻值为 0 或无穷大均表明绕组短路或断路

### 3.2 绝缘电阻检测

- 用兆欧表测量压缩机接线端子与压缩机外壳之间的绝缘电阻：
  - 正常值 > 2 MΩ
  - < 1 MΩ 表明绝缘老化或内部漏电
  - < 0.1 MΩ 表明严重漏电，需更换压缩机

### 3.3 运行电流检测

- 恢复供电，钳形电流表夹住压缩机供电线，测量运行电流：
  - 与额定电流对比，偏高表明负载过大或机械卡滞
  - 偏低或为零表明电路断路或启动器故障

## 4. 机械检测

- 用手转动压缩机尾部风扇叶片（断电状态下）：
  - 能轻松转动且无异响：机械部分基本正常
  - 卡顿、沉重或异响：内部机械部件（活塞、曲轴、阀片等）可能损坏
  - 完全无法转动：压缩机抱死，需更换

## 5. 启动器与保护器检测

- 测量启动器（继电器/PTC）线圈电阻是否正常
- 测量过载保护器是否导通（正常时导通，跳开时断开）
- 将启动器与保护器替换为已知正常的同型号部件进行排除法验证

## 6. 综合判断

| 现象 | 可能原因 |
|------|---------|
| 绕组电阻为 0 或短路 | 绕组烧毁，需更换压缩机 |
| 绕组电阻无穷大（断路） | 绕组断路，需更换压缩机 |
| 绝缘电阻过低 | 内部漏电，需更换压缩机 |
| 运行电流过大、外壳过热 | 机械卡滞或缺制冷剂（高负荷运行） |
| 压缩机不启动、有电流声 | 启动器或保护器故障，排查电气附件 |
| 压缩机不启动、无电流 | 主电源或控制电路故障 |
| 绕组正常、绝缘正常、无法转动 | 机械抱死，需更换压缩机 |
| 绕组正常、绝缘正常、电流正常但不制冷 | 压缩机阀片老化内漏 |

## 7. 安全注意事项

- 制冷系统内部带高压制冷剂，检测前严禁随意拆卸管路
- 压缩机内部有残余压力，维修时应缓慢泄压
- 电气检测前务必确认断电，且电容已充分放电（直接接触可能造成电击）
- 无制冷剂处理资质者请勿自行操作制冷剂环节
