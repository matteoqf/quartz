# 厄尔尼诺与拉尼娜现象

## 概述

厄尔尼诺（El Niño）和拉尼娜（La Niña）是太平洋赤道地区海水温度异常升降的现象，合称ENSO（厄尔尼诺-南方涛动）。它们是地球气候系统中最强的年际变化信号，对全球天气模式产生深远影响。

## 正常情况：沃克环流

在正常年份，太平洋赤道地区存在完整的沃克环流系统：

```
              西风
    ←─────────────────←
    
    印尼、澳大利亚           南美洲
    （暖池区）    太平洋    （冷水区）
       ↓↓         ↓↓         ↓↓
      上升         稳定        上升
      气流         降水        气流
    
         东风
    ──────────────────→
     （信风）
```

- **西太平洋暖池**：海水温度高（28-30°C），气流上升，降水充沛
- **东太平洋冷水区**：冷海水上涌，气流下沉，气候干燥
- **信风**：从东向西吹拂，推动暖水在西太平洋聚集

## 厄尔尼诺现象

### 形成机制

当信风异常减弱时，西太平洋的暖水向东回流，导致东太平洋海温异常升高：

```
              西风（减弱）
         ←──────────────←
         
    印尼、澳大利亚           南美洲
       ↓↓         ↓↓         ↓↑（异常暖）
      上升         上升        上升
                  ←←←暖水东移
         东风（减弱）
    ──────────────────→
```

### 诊断标准

- **海温异常**： Nino3.4区（西经120-170度，北纬5度-南纬5度）海温距平 ≥ +0.5°C，持续5个月以上
- **大气响应**：南方涛动指数（SOI）持续为负

### 气候影响

| 地区 | 影响 |
|------|------|
| 南美洲西岸 | 暴雨洪涝 |
| 印度尼西亚 | 干旱森林火灾 |
| 中国 | 暖冬、北方干旱、南方暴雨 |
| 北美 | 冬季偏暖 |

## 拉尼娜现象

### 形成机制

当信风异常增强时，西太平洋暖水更加堆积，东太平洋冷海水上涌更强：

```
              西风（增强）
         ←────────────────←
         
    印尼、澳大利亚           南美洲
    （超暖）     太平洋    （超冷）
       ↓↓↓        ↓↓↓        ↓↓↓
      强上升       强下沉      强下沉
      暴雨        干旱        干旱
         东风（增强）
    ──────────────────→
```

### 诊断标准

- **海温异常**：Nino3.4区海温距平 ≤ -0.5°C，持续5个月以上
- **大气响应**：南方涛动指数（SOI）持续为正

### 气候影响

| 地区 | 影响 |
|------|------|
| 中国 | 冷冬、台风增多 |
| 美国 | 北部严寒，南部干旱 |
| 澳大利亚 | 洪涝 |
| 南美洲 | 渔业增产（冷水中营养丰富） |

## 物理本质

### 海洋-大气耦合系统

厄尔尼诺与拉尼娜的本质是海洋与大气之间的正反馈耦合：

```
海温异常 → 气压变化 → 风场改变 → 海流改变 → 海温异常
    ↑                                      ↓
    └────────────── 反馈循环 ──────────────┘
```

### 关键反馈机制

1. **Bjerknes反馈**：海温梯度变化 → 信风强度变化 → 海流变化 → 海温变化
2. **赤道涌升流**：信风增强 → 冷海水上涌增强 → 海温降低 → 信风进一步增强
3. **季节性充电理论**：西太平洋暖水逐渐积累，在特定季节释放形成ENSO事件

### 能量来源

- 表面信风对海水的风应力做功
- 东西海温梯度储存的热能
- 海洋次表层温跃层存储的暖水

## 周期与预测

- **典型周期**：2-7年，平均约4年
- **事件持续时间**：通常9-12个月
- **预测方法**：海洋温跃层深度、海表温度、海洋声学、热力学模式

## 总结

厄尔尼诺与拉尼娜是海洋-大气耦合系统中的大规模振荡现象。它们的物理本质是赤道太平洋海温梯度的异常变化，通过沃克环流的调整影响全球气候。理解ENSO对于气候预测和防灾减灾具有重要意义。
