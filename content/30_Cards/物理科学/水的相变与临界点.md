# 水的相变与临界点

## 物质的三相

水是自然界中唯一能在常温下以固、液、气三态同时稳定存在的物质。这源于水分子特殊的氢键结构和其相图的独特形状。

水的三相：
- **冰（固态）**：水分子排列成晶体结构，通过氢键形成空间网格
- **水（液态）**：分子间氢键持续断裂重组，距离近但无固定取向
- **水蒸气（气态）**：分子间距大，氢键几乎完全断开

## 水的相图

相图是描述物质在不同温度、压力下存在相态的坐标图。水 的相图包含三条相变曲线，将平面划分为固、液、气三个区域。

```
        熔化曲线（固-液）
           /
          /
         /          气
        /          /
       /         /
      /        /
     *-------/------- 临界点 (374°C, 22.1 MPa)
    /        |
   /         |        液
  /          |
 /           |
/            |
-------------*
  熔点曲线   沸点曲线
```

### 三条关键曲线

1. **熔化曲线（固-液平衡线）**：冰与液态水的分界线
2. **沸腾曲线（液-气平衡线）**：液态水与水蒸气的分界线
3. **升华曲线（固-气平衡线）**：冰与水蒸气的分界线（在低压下）

三条曲线交汇于**三相点**，水在三相点的温度为 0.01°C，压力为 611.657 Pa。

## 基本相变过程

### 1. 熔化（Solid → Liquid）

固态冰吸收热量，晶体结构崩塌，分子获得流动性。

- **熔点**：在一个标准大气压下为 0°C
- **熔化热**：333.55 kJ/kg
- 熔化过程中温度保持不变，外界热量完全用于克服分子间作用力

### 2. 凝固（Liquid → Solid）

熔化的逆过程，液态水释放热量，分子重新排列成晶体结构。

- **凝固点**：0°C（与熔点相同，热力学平衡）
- 凝固时体积膨胀约 9%，这是冰能浮在水面的原因

### 3. 汽化（Liquid → Gas）

液态水转化为气态水蒸气。有两种方式：

**蒸发**：发生在液体表面，温度低于沸点时持续进行。
- 速率受温度、湿度、表面积、通风影响
- 是生命体散热的主要方式（出汗）

**沸腾**：在液体内部剧烈汽化，必须达到沸点且持续吸热。

- **沸点**：一个标准大气压下为 100°C
- **汽化热**：2260 kJ/kg（比熔化热高得多）
- 沸点随压力升高而升高（高压锅原理）

### 4. 液化（Gas → Liquid）

水蒸气遇冷或被压缩时转化为液态水。

- 释放汽化热（凝结热）
- 云、雾、雨的形成都是液化过程
- 空气达到饱和时水蒸气凝结成水滴

### 5. 升华（Solid → Gas）

固态冰不经过液态直接变为气态。

- 条件：低温、低压（如高海拔或干燥环境）
- **升华热**：2838 kJ/kg（等于熔化热 + 汽化热）
- 干冰（固体CO₂）是最常见的升华例子
- 冬季结冰的衣服变干就是升华

### 6. 凝华（Gas → Solid）

气态水蒸气直接变成冰，逆升华。

- 条件：骤然降温（如霜、雪的形成）
- 凝华放热

## 临界点与超临界流体

### 临界温度与临界压力

每种物质都有一个**临界温度**（Tc）和**临界压力**（Pc）：

- **临界温度**：物质能够液化的最高温度。高于此温度，无论压力多大，气体都不能被液化
- **临界压力**：在临界温度下使气体液化所需的最小压力
- **临界点**：(Tc, Pc) 这个唯一点

### 水的临界参数

| 参数 | 数值 |
|------|------|
| 临界温度 | 373.946°C (647.096 K) |
| 临界压力 | 22.064 MPa (约 218 atm) |
| 临界密度 | 322 kg/m³ |

当温度和压力同时超过临界点时，物质进入**超临界流体**状态。

### 超临界水的特性

在临界点以上，水呈现一种既非液态也非气态的特殊状态：

- **密度**：可从类似气体连续调节到类似液体
- **扩散系数**：高于液体，接近气体
- **粘度**：远低于液体，流动性极好
- **介电常数**：急剧下降，从 ~80（室温水）降到 ~5（超临界水）

超临界水具有独特的溶解能力和反应性，广泛应用于：
- 超临界水氧化法处理有机废弃物
- 从植物中提取天然产物
- 地质过程中的流体作用

## 相变的微观机制

### 一级相变 vs 二级相变

| 特性 | 一级相变 | 二级相变 |
|------|----------|----------|
| 体积 | 突变 | 连续 |
| 熵 | 突变 | 连续 |
| 热容 | 无限大（峰值） | 突变 |
| 潜热 | 有 | 无 |
| 例子 | 熔化、沸腾 | 超导转变、磁性转变 |

水的相变（熔化、沸腾、凝结、凝固）均为**一级相变**，伴有潜热和体积突变。

### 相变的热力学描述

相变可用 **Clausius-Clapeyron 方程**描述：

$$\frac{dP}{dT} = \frac{L}{T \Delta v}$$

其中：
- $L$ 为相变潜热
- $\Delta v$ 为相变时的体积变化
- $T$ 为相变温度

该方程给出了相平衡曲线斜率与潜热、体积变化的关系。

## 水的异常性质

水有诸多反常特性，与氢键网络结构密切相关：

1. **密度异常**：4°C时密度最大，而非0°C；冰的密度比水低9%
2. **熔点随压力反常**：压力增加使冰的熔点降低（大多数物质相反）
3. **沸点异常高**：比同族氢化物（ H₂S、H₂Se）沸点高得多
4. **比热容大**：水是常见物质中比热容最大的
5. **蒸发潜热高**：使水成为优秀的冷却介质

这些异常性深刻影响地球气候、生态系统和生命活动。

## 生活中的相变现象

| 现象 | 相变类型 |
|------|----------|
| 冰浮在水面 | 凝固（体积膨胀） |
| 汗水蒸发降温 | 蒸发（汽化） |
| 冬天湖面结冰 | 凝固 |
| 雾和云 | 凝结（液化） |
| 霜的形成 | 凝华 |
| 干衣机烘干衣物 | 蒸发 |
| 冰箱制冷 | 汽化-液化循环 |

## 关键启示

1. 水的三相平衡揭示了热力学相变的一般规律
2. 临界点是物质状态的极限，高于此点液气不分
3. 水的相变在工程、生命、环境等领域有广泛应用
4. 相变潜热是能量储存和传递的重要形式
5. 水的异常性质源于氢键，这一微观结构决定了宏观行为
