# 🎭 社会工程学反制

## 电信诈骗脚本逻辑拆解

### 典型诈骗剧本结构

```
第一阶段：建立信任
  ├── 准确报出你的个人信息（已泄露数据库）
  ├── 冒充权威身份（公安局/快递/银行）
  └── 通过长期"培养感情"（杀猪盘）或"制造恐慌"（冒充公检法）

第二阶段：制造时间压迫感
  ├── "今天必须处理，否则账户冻结"
  ├── "限时30分钟，超时将承担法律责任"
  └── 目的：不给你思考/求助/核实的时间

第三阶段：切断退路
  ├── 要求保密（"这是机密，不能告诉任何人"）
  ├── 要求转移资金到"安全账户"
  └── 要求卸载反诈APP / 开启屏幕共享（监控你操作）

第四阶段：收割
  └── 资金到账 → 立即分散提走 → 消失
```

### 反制核心原则

```
┌─────────────────────────────────────────┐
│  所有让你"立刻/马上/不能等"的 → 暂停      │
│  所有让你"不能告诉任何人"的 → 警惕        │
│  所有让你转账到"安全账户"的 → 100%诈骗   │
│  验证码 = 你的钱，不给任何人             │
└─────────────────────────────────────────┘
```

## 杀猪盘：资金养护周期

### 什么是杀猪盘

```
杀猪盘 = 婚恋/交友诈骗
  "猪" = 受害者
  "养" = 建立感情信任的过程（数周到数月）
  "杀" = 诱导投资/转账

特点：
  → 伪装成高富帅/白富美
  → 感情培养期极长（温水煮青蛙）
  → 感情稳固后才引入"投资机会"
  → 通常先让你小赚一笔 → 诱导加大投入 → 血本无归
```

### 识别信号

```
⚠️ 杀猪盘早期信号：
├── 偶然相识，但"恰好"是单身高管/军人/海归
├── 频繁主动找你，但从不视频（"信号不好"）
├── 聊1-2周后开始提"副业/投资/漏洞"
├── 展示"投资收益截图"（PS伪造）
├── 始终不告诉你真实姓名/工作单位
└── 关系发展过快（迅速建立亲密感）
```

## 钓鱼邮件（Phishing）分析

### 发件人伪装剥离

```
钓鱼邮件常见伪装：

伪装的真实发件人：
  ✅ apple@apple.com              （真的）
  ❌ apple-support@mail-apple.com （钓鱼）
  ❌ apple@applesupport.top       （钓鱼）
  ❌ Apple <no-reply@apple-id.com> （注意括号内域名）

域名分析技巧：
  → apple.com 是Apple
  → apple.com.malignsite.com 是malignsite
  → apple-support.vip 没有任何大公司用这种域名

查看真实发件人：
  → 网页邮箱：鼠标悬停显示真实地址
  → 手机邮件：点击发件人 → 展开详情
```

### URL结构分析

```
钓鱼URL常见模式：

恶意短链接（隐藏真实地址）：
  → t.cn/xxxxx → 需打开前核查（t.cn是微博短链）
  → bit.ly/xxxxx → 同上

伪装的真实URL：
  → paypa1.com（注意是数字1不是字母l）
  → applе.com（注意是西里尔字母а不是拉丁a）

真实URL结构：
  https://www.paypal.com/myaccount/转账
  协议://子域名.主域名.顶级域名/路径

域名注册查询（whois）：
  → 查询域名注册时间（新注册域名风险高）
  → 查询注册人信息（隐私保护=无法核实）
```

## 深度伪造（Deepfake）识别

### 视觉痕迹

```
⚠️ 视频 Deepfake 识别点：
├── 眼睛/眨眼：早期AI眨眼频率/幅度异常
├── 面部边缘：头发与面部交界处模糊/抖动
├── 光线不一致：面部反射与背景光线不匹配
├── 眨眼皮细节：眼睑边缘不自然
├── 牙齿/口腔：说话时牙齿形态不自然
└── 表情过渡：大笑/大哭等极端表情时失真

音频 Deepfake 识别点：
├── 语调机械感（缺乏真人情感起伏）
├── 背景噪音模式异常
├── 特定词语重音不自然
└── 录制质量异常（过高保真度）
```
