# 🏛️ 行政官僚体系对接

## 个人档案流转逻辑

### 档案类型与流转

```
档案类型：
├── 学籍档案（上学期间）
│    └── 高中/大学/研究生期间的学业/奖惩记录
├── 人事档案（工作后）
│    ├── 工作经历/职称/调动
│    ├── 工资级别/社保缴纳记录
│    └── 党员材料（如果有）
└── 党员档案（单独管理）
     └── 与人事档案分离

流转节点：
  高中毕业 → 档案转至户籍地人才市场/街道
  大学入学 → 档案转至大学
  大学毕业后 → 档案转至：
               ├── 工作单位（有档案保管资质）
               ├── 工作地人才市场（单位无资质）
               └── 户籍地人才市场/街道（未就业）
```

### 档案常见问题

```
⚠️ 档案在自己手里（死档）：
  → 超过择业期（通常2年）档案在自己手里
  → 人才市场不接收个人自带档案
  → 解决：找原学校走公对公渠道转递

⚠️ 档案丢失：
  → 影响：无法办理职称/落户/养老保险
  → 解决：从原学校/单位逐项补办材料
  → 无法补办的：出具证明 + 说明

⚠️ 档案信息错误：
  → 出生日期/姓名不一致
  → 解决：持原始证件（户口本/毕业证）申请更正
```

## 街道办/居委会 vs 派出所

### 管辖权区分

```
街道办/居委会（基层自治组织）：
├── 民政事务：低保/老龄/残疾补贴申请
├── 计划生育：准生证/生育服务证办理
├── 物业协调：业主大会/物业纠纷调解
├── 人口统计：常住/流动人口登记
├── 综治维稳：群防群治/治安巡逻
└── 党团关系：党组织关系转接/党员活动

公安派出所（公安机关派出机构）：
├── 户籍管理：落户/迁移/身份证办理
├── 治安管理：治安案件/轻微刑事案件
├── 人口管理：出租屋登记/境外人员管理
├── 刑事侦查：立案后的侦查工作
├── 110出警：突发事件/治安求助
└── 特种行业：旅馆/网吧/印章刻制备案
```

### 找谁办事

```
想办事？先判断这是"行政事务"还是"公安事务"：

行政事务（找街道办/居委会）：
  → 申请补贴/证明/登记/协调

公安事务（找派出所）：
  → 身份证/户籍/治安案件/报警

注意：
  ├── 两者不是一个系统，不能互相替代
  ├── 街道办/居委会是"自治组织"，没有强制权
  ├── 派出所有执法权，但通常只处理涉法事务
  └── 日常民生事务找街道办效率更高
```

## 证件遗失的止损流程

### 紧急止损步骤

```
第一步：立即挂失（阻止他人冒用）
  ├── 身份证 → 就近派出所/公安局
  │           → 登报声明（非必须，但建议）
  ├── 银行卡 → 电话银行客服挂失（立即）
  │           → 24小时内到网点补办
  ├── 手机卡 → 运营商营业厅挂失
  │           → 防止验证码登录你的账户
  └── 社保卡 → 人社局/社保中心挂失
               → 补办周期：7-30天

第二步：补办证件
  ├── 身份证：派出所，60天制发（加急更快）
  ├── 银行卡：网点，7-10个工作日
  ├── 驾照：车管所
  └── 护照：出入境管理局

第三步：关联账户安全检查
  ├── 银行卡：确认无异常交易
  ├── 支付宝/微信：解绑旧手机/重新绑定
  ├── 常用网站：检查异常登录记录
  └── 通知银行/平台：新的证件信息更新
```

### 登报声明的作用

```
登报声明 ≠ 挂失，但有以下作用：
  → 产生公示效力（公众有义务注意）
  → 万一有人冒用，有证据证明你已尽止损义务
  → 部分机构（房产/车辆）办理业务时可能要求

现代做法：
  → 身份证：大多数地区已无需登报（公安系统联网）
  → 房产证/土地证：建议登报
  → 营业执照：必须登报声明
```
