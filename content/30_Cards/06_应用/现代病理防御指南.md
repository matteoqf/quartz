# 🦠 现代病理防御指南

## HPV疫苗与防护机制

### HPV是什么

```
人乳头瘤病毒（Human Papillomavirus）
  → 超过200种亚型
  → 高危型（16/18/45/52/58等）→ 宫颈癌、肛门癌、阴茎癌
  → 低危型（6/11等）→ 尖锐湿疣（生殖器疣）
  → 主要传播途径：性接触（避孕套不能完全预防）
```

### 疫苗类型与接种时间轴

| 疫苗 | 预防亚型 | 适用年龄 | 接种剂次 |
|------|---------|---------|---------|
| 2价 | 16/18（致癌率最高） | 9-45岁 | 0/1/6月（3针） |
| 4价 | 6/11/16/18 | 20-45岁 | 0/2/6月（3针） |
| 9价 | 6/11/16/18/31/33/45/52/58 | 16-26岁 | 0/2/6月（3针） |

### 关键事实

```
✅ 9-14岁接种效果最佳（尚未性接触，免疫应答最强）
✅ 有性生活后仍可接种（虽效果下降，仍有保护）
✅ 已有HPV感染 → 接种后仍可预防其他亚型
✅ 接种后仍需定期宫颈筛查（TCT+HPV联合检测）
✅ 男性可接种4价/9价（预防肛门癌/阴茎癌/尖锐湿疣）
```

## 幽门螺旋杆菌（Hp）

### 传播途径与风险

```
幽门螺旋杆菌：
  → 胃部生存的细菌（胃酸杀不死）
  → 感染全球约50%人口（中国约60%）
  → 传播途径：口-口（接吻/共用餐具）、粪-口（食物水源）

致癌路径：
  幽门螺旋杆菌感染 → 慢性胃炎 → 萎缩性胃炎 → 肠化生 → 胃癌
  （WHO列为I类致癌物）
```

### 检测与根除

```
检测方法：
├── 碳13/碳14呼气试验（最常用，无创）
├── 粪便抗原检测
├── 胃镜活检（最准确）
└── 血液抗体检测（仅提示曾感染，不能区分现感染）

根除方案：
├── 四联疗法：质子泵抑制剂（PPI）+ 铋剂 + 两种抗生素
├── 疗程：14天（必须吃完整个疗程）
├── 根除率：约80-90%（耐药率影响效果）
└── 根除后有复发可能（再感染/复发）
```

## 久坐与静脉血栓（DVT）

### 机制

```
久坐不动 →
  小腿肌肉不收缩 →
  血液回流减慢 →
  血液瘀滞 →
  凝血因子聚集 →
  深静脉血栓（DVT）

典型场景：
├── 长时间飞机经济舱（"经济舱综合征"）
├── 长途汽车/火车
├── 办公室久坐
└── 骨折/手术后长期卧床
```

### 识别信号

```
⚠️ 下肢深静脉血栓信号：
├── 单侧下肢肿胀（左右腿粗细不一）
├── 患肢疼痛（尤其是活动后加重）
├── 皮肤发红/发热
└── 浅静脉曲张突然加重

🚨 肺栓塞（血栓脱落跑到肺部）信号：
├── 胸痛 + 呼吸困难
├── 咯血
└── 意识丧失
→ 立即拨打120！
```

### 预防措施

```
┌─────────────────────────────────────────┐
│  每坐1小时 → 站起来活动5-10分钟          │
│  坐时做"踝泵运动"：脚尖勾起/放下，重复   │
│  长途旅行每2-3小时起来活动              │
│  多喝水（脱水会增加血液粘稠度）          │
│  弹力袜（长途飞行/高危人群）            │
└─────────────────────────────────────────┘
```

## 常见致癌物分类与规避

### WHO/IARC分类

```
1类致癌物（明确致癌）：
├── 吸烟/二手烟
├── 酒精（含乙醇）
├── 加工肉制品（香肠/培根）
├── 中式咸鱼
├── 黄曲霉素（发霉粮食/花生油）
├── 幽门螺旋杆菌（Hp）
├── HPV高危亚型
└── 苯并芘（烧烤/烟熏食品）

2A类（很可能致癌）：
├── 红肉（过量摄入）
├── 65℃以上热饮（食道癌）
└── 夜班工作（生物钟紊乱）

2B类（可能致癌）：
├── 腌菜（亚硝酸盐）
└── 汽车尾气
```

### 物理规避策略

```
黄曲霉素：
  ❌ 发霉花生/玉米/坚果 → 扔掉（整包扔）
  ✅ 正规渠道购买，注意保质期
  ✅ 久放食品定期检查

亚硝酸盐/腌菜：
  ❌ 长期大量食用腌菜
  ✅ 新鲜蔬菜为主，腌菜偶尔吃

苯并芘（烧烤）：
  ❌ 过度食用明火烧烤
  ✅ 燃气烤架优于炭火
  ✅ 食材不直接接触火焰

幽门螺旋杆菌：
  ✅ 分餐制/使用公筷
  ✅ 餐具定期消毒
  ✅ 高危人群定期检测
```
