# 🧠 心理危机干预

## 躯体化症状的识别

### 什么是躯体化

> 心理痛苦通过身体症状表现出来，但身体检查查不出器质性病变。**不是装病，是真实的痛苦**。

### 常见模式

```
压力/焦虑/抑郁 → 躯体化症状：

消化系统：
├── 肠胃痉挛（紧张时胃痛/腹泻）
├── 肠易激综合征（IBS）
└── 食欲改变（暴食或厌食）

心血管系统：
├── 心悸（感觉心跳很快/很强）
├── 胸闷/胸痛（但心电图正常）
└── 血压一过性升高

肌肉紧张：
├── 肩颈僵硬/头痛（紧张性头痛）
├── 颞下颌关节紊乱（磨牙/牙关紧咬）
└── 慢性腰背痛

呼吸系统：
├── 胸闷（感觉喘不上气）
└── 过度换气（呼吸急促导致手脚发麻）
```

### 识别要点

```
关键信号：身体症状与检查结果不匹配
  └── 症状严重，但抽血/影像/心电图都正常
  └── 症状随情绪波动（焦虑加重时症状加重）
  └── 多种系统同时出现症状（胃痛+心悸+头痛）
```

## 惊恐发作（Panic Attack）

### 什么是惊恐发作

```
急性焦虑的极端形式：
  → 突然出现强烈恐惧感（失控感/濒死感）
  → 伴随强烈躯体症状
  → 通常在10分钟内达到高峰
  → 通常持续20-30分钟自行缓解
```

### 症状清单

```
⚠️ 至少4项：
├── 心悸/心跳强烈
├── 出汗
├── 颤抖
├── 呼吸急促/过度换气
├── 窒息感
├── 胸痛
├── 恶心/腹部不适
├── 头晕/站不稳
├── 手脚发麻/刺痛
├── 寒颤或潮热
├── 现实解体（感觉不真实）
└── 失控/发疯感
```

### 呼吸对抗法

```
当你感到恐慌时：
1. 识别过度换气（呼吸急促 → 排出过多CO2 → 血管收缩 → 手脚发麻）

2. 对抗方法——纸袋/塑料袋呼吸法：
   ├── 用纸袋罩住口鼻（不要完全密封）
   ├── 缓慢吸气（用鼻子，4秒）
   ├── 缓慢呼气（用嘴，6-8秒）
   ├── 重复5-10次
   └── 目的：把CO2浓度补回来

3. 无纸袋时——4-7-8呼吸法：
   ├── 吸气（用鼻子，4秒）
   ├── 屏住呼吸（7秒）
   ├── 呼气（用嘴，8秒）
   └── 重复3-4次

4. 接地技术（Grounding）：
   ├── 说出5样你能看到的东西
   ├── 说出4样你能触摸的东西
   ├── 说出3样你能听到的声音
   ├── 说出2样你能闻到的气味
   └── 说出1样你能尝到的味道
```

## 抑郁症的临床诊断

### 诊断标准（DSM-5简化版）

> 连续2周，出现**5个以上症状**，且**至少包括1号和2号症状**：

| 编号 | 症状 | 说明 |
|------|------|------|
| ① | 情绪低落 | 几乎整天都悲伤/空虚/绝望 |
| ② | 兴趣减退 | 对所有/几乎所有活动丧失兴趣 |
| ③ | 体重变化 | 显著减轻/增加（1个月内>5%） |
| ④ | 睡眠障碍 | 失眠（难入睡/早醒）或嗜睡 |
| ⑤ | 精神运动 | 迟滞（动作慢/说话慢）或激越 |
| ⑥ | 疲劳 | 精力耗尽感 |
| ⑦ | 无价值感 | 过度/不合理的内疚感 |
| ⑧ | 注意力障碍 | 决策困难/犹豫不决 |
| ⑨ | 自杀念头 | 反复出现死亡/自伤想法 |

### "心情低落" vs 抑郁症

| 维度 | 心情低落（正常反应） | 抑郁症（需要干预） |
|------|-----------------|-----------------|
| 诱因 | 有具体事件（失恋/失败） | 可能无明显诱因 |
| 持续 | 数小时至数天 | ≥ 2周 |
| 功能影响 | 轻微影响生活 | 严重影响工作/社交/自理 |
| 症状数量 | 1-2个情绪症状 | ≥ 5个情绪+躯体症状 |
| 晨重夕轻 | 不明显 | 典型特征（早晨最重） |

> ⚠️ 如果你连续2周以上感觉"无法正常工作/生活/社交"，建议就医。

## 何时寻求专业帮助

```
建议就医/心理咨询的信号：
├── 症状持续超过2周不缓解
├── 有自杀念头或自我伤害冲动
├── 严重影响工作/社交/家庭功能
├── 伴随物质滥用（酗酒/吸毒）
├── 出现幻觉/妄想（现实检验能力丧失）
└── 家人朋友明显观察到性格大变
```
