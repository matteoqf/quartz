# 婚前财产与婚后财产的界定

## 核心原则

- **婚前财产**：一方在结婚登记前已取得的财产
- **婚后财产**：婚姻关系存续期间一方或双方取得的财产
- **约定优先**：夫妻双方对财产归属有约定的，按约定处理

## 婚前财产认定

### 范围

| 类型 | 说明 |
|------|------|
| 个人劳动所得 | 结婚登记前因劳动、经营、职务等获得的财产 |
| 继承或受赠财产 | 遗嘱或赠与合同中确定只归一方的财产 |
| 专用生活用品 | 结婚前购买供本人使用的物品（价值较大的除外） |
| 婚前财产的孳息 | 存款利息、股票分红等（需为婚前本金所生） |
| 工伤赔偿等 | 结婚前因人身伤害获得的赔偿 |

### 关键时点

- **婚姻登记日期**为分界线：登记之前取得的财产为婚前财产
- 婚后才进行产权登记的财产，一般认定为婚后财产（除非有充分证据证明实际出资在婚前）

## 婚后财产认定

### 范围

| 类型 | 说明 |
|------|------|
| 工资、奖金 | 任何一方的工资、奖金收入 |
| 生产、经营收益 | 包括个体工商户、有限责任公司等经营所得 |
| 知识产权收益 | 婚后实际取得或明确可以取得的知识产权收益 |
| 继承或受赠财产 | 遗嘱或赠与合同中未明确归一方的财产 |
| 住房补贴、住房公积金 | 婚姻关系存续期间取得的 |
| 养老保险金、失业保险金 | 达到法定退休年龄前缴纳的部分 |
| 投资收益 | 婚后使用共同财产投资获得的收益 |

### 特殊情况

- **婚后一方父母出资购买不动产**：若无明确约定，认定为对夫妻双方的赠与（2021年《民法典》实施后）
- **婚后一方继承的财产**：若继承人未明确表示只归一方，则为夫妻共同财产

## 财产混同与转化

### 常见混同情形

- 婚前存款与婚后收入混同使用，难以区分
- 婚前购婚后还贷的房产，婚后共同还贷部分及对应增值为共同财产
- 婚前财产婚后出租所得租金（可能被认定为共同财产）

### 举证责任

- 主张财产为婚前个人财产的一方承担举证责任
- 建议：保留婚前财产凭证、银行流水、购房合同等证据

## 法律依据

- **《中华人民共和国民法典》** 第一千零六十二条、第一千零六十三条
- **《最高人民法院关于适用〈中华人民共和国民法典〉婚姻家庭编的解释（一）》**

## 注意事项

1. **及时约定**：大额财产可在婚前或婚后通过书面协议约定归属
2. **保留证据**：保存好财产取得的时间节点、支付凭证等
3. **混同风险**：避免婚前个人财产与共同财产大规模混同
4. **父母出资**：购房等大额支出建议明确出资性质（借款/赠与）
