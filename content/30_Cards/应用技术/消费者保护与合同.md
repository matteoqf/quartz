# 🛒 消费者保护与合同

## 七天无理由退货

### 适用范围

```
✅ 可退货（网络/电话/邮购）：
├── 所有商品（除以下排除）
└── 收到商品7天内可申请

❌ 不适用七天无理由：
├── 定制品（专为您定制）
├── 鲜活易腐（生鲜水果等）
├── 数字化商品（已下载的音乐/软件）
├── 报纸期刊（时效性强）
├── 贴身衣物（内衣裤等）
└── 拆封后影响二次销售的商品
```

### 退货流程

```
1. 在退货时限内（签收次日起7天）申请
2. 保证商品完好（不影响二次销售）
3. 经营者收到退货商品后7日内退款
4. 退货费用：有质量问题/发错货 → 经营者承担
            无理由退货 → 消费者承担运费
```

## "退一赔三"与"退一赔十"

### 欺诈消费的"退一赔三"

> 《消费者权益保护法》第55条：经营者提供商品有欺诈行为 → 退一赔三，赔一不足500按500算。

```
欺诈认定（经营者明知故犯）：
├── 掺杂掺假、以次充好
├── 虚假标注（产地/成分/功能）
├── 虚假宣传（功效与实际不符）
└── 伪造产地/冒用他人厂名厂址

计算示例：
  购买商品 ¥1,000 → 退 ¥1,000 + 赔 ¥3,000 = ¥4,000
```

### 食品安全的"退一赔十"

> 《食品安全法》第148条：生产/经营不符合食品安全标准的食品 → 退一赔十，赔一不足1000按1000算。

```
十倍赔偿适用：
├── 过期食品
├── 添加剂超标
├── 含有禁止添加的物质
├── 生产日期/保质期虚假标注
└── 原料不符合国家标准

计算示例：
  购买问题食品 ¥50 → 退 ¥50 + 赔 ¥500（十倍不足1000，按1000算）
```

## 租房合同核心条款

### "不可抗力"条款

```
不可抗力 = 不能预见 + 不能避免 + 不能克服
          典型场景：地震、台风、战争、政府征用

租房中的不可抗力处理：
├── 因不可抗力导致合同无法履行 → 双方均不违约
├── 租金：实际居住天数折算，未住期间租金退还
└── 解除合同：通知对方即可，无需赔偿违约金
```

### "买卖不破租赁"原则

> 即使房东卖房，新业主也不能赶走已签租约的租客。

```
法律效果：
├── 租赁合同对新业主继续有效
├── 租客有权住到租约到期
├── 新业主取得租金收益
└── 原房东有义务配合租客继续履行

例外（不可对抗的情形）：
├── 先租后抵押（抵押权设立在租赁之前）
└── 法院查封后拍卖
```

### 租房防坑清单

```
签合同前必查：
□ 房产证原件 + 房东身份证（确认真实房东）
□ 确认租金包含哪些费用（物业/水电/网络）
□ 押金退还条件（有无折旧费扣除条款）
□ 提前解约条款（违约金是否合理）
□ 房屋设施清单（入住前拍照留证）
□ 违约责任是否对等（不能只有租客违约条款）
```
