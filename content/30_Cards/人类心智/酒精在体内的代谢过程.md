# 酒精在体内的代谢过程

## 概述

酒精（乙醇）是一种小分子物质，摄入后无需消化，直接通过胃壁和肠壁吸收进入血液。约20%由胃吸收，80%由小肠吸收，2-5分钟内即可进入血液。

## 主要代谢场所：肝脏

肝脏是酒精代谢的唯一场所，占总代谢量的90%以上。

## 代谢过程

### 第一阶段：乙醇 → 乙醛

**关键酶：乙醇脱氢酶（ADH）**

- 位置：肝细胞细胞质
- 辅助因子：NAD⁺（烟酰胺腺嘌呤二核苷酸）
- 反应：乙醇 + NAD⁺ → 乙醛 + NADH + H⁺
- 特点：速度较慢，是限速步骤

### 第二阶段：乙醛 → 乙酸

**关键酶：乙醛脱氢酶（ALDH）**

- 位置：肝细胞线粒体
- 辅助因子：NAD⁺
- 反应：乙醛 + NAD⁺ + H₂O → 乙酸 + NADH + H⁺
- 特点：速度较快

### 第三阶段：乙酸 → 乙酰辅酶A

- 乙酸进入血液循环
- 在外周组织中转化为乙酰辅酶A
- 进入三羧酸循环（TCA循环）彻底氧化为CO₂和H₂O

## 关键点

1. **NAD⁺的消耗**：酒精代谢大量消耗NAD⁺，影响糖异生和脂肪代谢
2. **乙醛的毒性**：乙醛是导致宿醉、脸红、心悸的主要原因
3. **个体差异**：ALDH酶活性差异导致酒精耐受性不同（亚洲人常见ALDH2缺陷）
4. **代谢速率**：平均每小时代谢7-10克纯酒精（约一标准杯）

## 影响因素

| 因素 | 影响 |
|------|------|
| 性别 | 女性胃内ADH活性较低，吸收更快 |
| 体重 | 体重越大，体液稀释越多，浓度越低 |
| 进食 | 空腹时吸收更快更完全 |
| 基因 | ADH和ALDH基因多态性影响代谢速度 |
| 肝脏功能 | 肝病患者代谢能力显著下降 |
