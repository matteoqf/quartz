# 队列与栈的实现与应用场景

## 1. 栈（Stack）

### 1.1 概念

栈是一种**后进先出（LIFO, Last In First Out）**的线性数据结构。只允许在栈顶进行插入（push）和删除（pop）操作。

### 1.2 基本操作

| 操作 | 说明 | 时间复杂度 |
|------|------|-----------|
| push | 向栈顶添加元素 | O(1) |
| pop | 移除栈顶元素并返回 | O(1) |
| top/peek | 查看栈顶元素 | O(1) |
| isEmpty | 判断栈是否为空 | O(1) |
| size | 返回栈中元素数量 | O(1) |

### 1.3 Python 实现

```python
class Stack:
    def __init__(self):
        self._data = []
    
    def push(self, item):
        self._data.append(item)
    
    def pop(self):
        if self.is_empty():
            raise IndexError("pop from empty stack")
        return self._data.pop()
    
    def peek(self):
        if self.is_empty():
            raise IndexError("peek from empty stack")
        return self._data[-1]
    
    def is_empty(self):
        return len(self._data) == 0
    
    def size(self):
        return len(self._data)
```

### 1.4 应用场景

- **函数调用栈**：程序运行时管理函数调用关系
- **表达式求值**：中缀转后缀/前缀表达式
- **括号匹配**：检验字符串中的括号是否成对出现
- **深度优先搜索（DFS）**：图的遍历
- **撤销/回退功能**：编辑器的 Ctrl+Z
- **浏览器前进/后退**：历史记录管理

## 2. 队列（Queue）

### 2.1 概念

队列是一种**先进先出（FIFO, First In First Out）**的线性数据结构。在队尾添加元素，在队首删除元素。

### 2.2 基本操作

| 操作 | 说明 | 时间复杂度 |
|------|------|-----------|
| enqueue | 向队尾添加元素 | O(1) |
| dequeue | 移除队首元素并返回 | O(1) |
| front | 查看队首元素 | O(1) |
| rear | 查看队尾元素 | O(1) |
| isEmpty | 判断队列是否为空 | O(1) |
| size | 返回队列中元素数量 | O(1) |

### 2.3 Python 实现（基础版）

```python
class Queue:
    def __init__(self):
        self._data = []
    
    def enqueue(self, item):
        self._data.append(item)
    
    def dequeue(self):
        if self.is_empty():
            raise IndexError("dequeue from empty queue")
        return self._data.pop(0)
    
    def front(self):
        if self.is_empty():
            raise IndexError("front from empty queue")
        return self._data[0]
    
    def is_empty(self):
        return len(self._data) == 0
    
    def size(self):
        return len(self._data)
```

### 2.4 Python 实现（高效版，使用 collections.deque）

```python
from collections import deque

class EfficientQueue:
    def __init__(self):
        self._data = deque()
    
    def enqueue(self, item):
        self._data.append(item)
    
    def dequeue(self):
        if self.is_empty():
            raise IndexError("dequeue from empty queue")
        return self._data.popleft()
    
    def front(self):
        if self.is_empty():
            raise IndexError("front from empty queue")
        return self._data[0]
    
    def is_empty(self):
        return len(self._data) == 0
    
    def size(self):
        return len(self._data)
```

### 2.5 应用场景

- **广度优先搜索（BFS）**：图的层级遍历
- **任务调度**：按顺序处理任务（如打印机队列）
- **消息队列**：异步通信、任务解耦
- **缓存淘汰策略**：如 FIFO
- **约瑟夫斯问题**：经典概率问题
- **按序处理请求**：网络请求、线程池任务分配

## 3. 特殊队列

### 3.1 双端队列（Deque）

两端都可以进行插入和删除操作。

```python
from collections import deque

dq = deque()
dq.appendleft(1)  # 左侧添加
dq.append(2)      # 右侧添加
dq.pop()          # 右侧删除
dq.popleft()      # 左侧删除
```

**应用场景**：滑动窗口最大值、撤销操作、LRU 缓存

### 3.2 优先队列（Priority Queue）

元素按优先级顺序出队，而非 FIFO。

```python
import heapq

pq = []
heapq.heappush(pq, (3, "task3"))
heapq.heappush(pq, (1, "task1"))
heapq.heappush(pq, (2, "task2"))

heapq.heappop(pq)  # 返回 (1, "task1")
```

**应用场景**：Dijkstra 最短路径、Huffman 编码、任务调度（高优先级先执行）

## 4. 栈与队列的对比

| 特性 | 栈 | 队列 |
|------|-----|------|
| 原则 | LIFO | FIFO |
| 主要操作 | push/pop | enqueue/dequeue |
| 应用重点 | 回溯、递归模拟 | 顺序处理、缓冲 |
| 图算法 | DFS | BFS |

## 5. 典型算法示例

### 5.1 括号匹配（栈）

```python
def is_valid(s: str) -> bool:
    stack = []
    mapping = {')': '(', ']': '[', '}': '{'}
    
    for char in s:
        if char in mapping:
            top = stack.pop() if stack else '#'
            if mapping[char] != top:
                return False
        else:
            stack.append(char)
    
    return not stack

# 测试
print(is_valid("()[]{}"))  # True
print(is_valid("([)]"))    # False
```

### 5.2 岛屿数量（队列 BFS）

```python
from collections import deque

def num_islands(grid):
    if not grid:
        return 0
    
    rows, cols = len(grid), len(grid[0])
    islands = 0
    
    for r in range(rows):
        for c in range(cols):
            if grid[r][c] == '1':
                islands += 1
                grid[r][c] = '0'
                q = deque([(r, c)])
                
                while q:
                    cr, cc = q.popleft()
                    for dr, dc in [(0,1),(0,-1),(1,0),(-1,0)]:
                        nr, nc = cr + dr, cc + dc
                        if 0 <= nr < rows and 0 <= nc < cols and grid[nr][nc] == '1':
                            grid[nr][nc] = '0'
                            q.append((nr, nc))
    
    return islands
```

## 6. 总结

- **栈**：适合需要"最近先处理"场景，如函数调用、括号匹配、DFS
- **队列**：适合需要"先来先服务"场景，如任务调度、BFS、缓冲
- **选择依据**：根据问题的数据访问顺序需求选择合适的数据结构
