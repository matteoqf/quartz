# ☠️ 化学危险与材料科学

## 致命毒气合成禁忌

### 84消毒液 + 洁厕灵 = 氯气

```
化学反应：
  次氯酸钠（84主要成分）+ 盐酸（洁厕灵主要成分）
  → 产生氯气（Cl₂）

氯气危害：
  → 刺激呼吸道黏膜（咳嗽/呼吸困难）
  → 吸入高浓度 → 肺水肿/窒息/死亡
  → 国防军事用化学武器级别

正确清洁顺序：
  ① 先用洁厕灵，用水冲净
  ② 再用84消毒液
  ③ 两者绝对不能同时使用
  ④ 清洁时保持通风
```

### 其他危险组合

| 组合 | 产物 | 危害 |
|------|------|------|
| 84 + 醋 | 氯气（少量） | 呼吸道刺激 |
| 84 + 洁厕灵 | 氯气（大量） | 致命 |
| 洁厕灵 + 管道疏通剂 | 硫化氢（剧毒） | 吸入即死 |
| 酒精 + 84 | 氯仿（中等毒性） | 损伤肝脏 |

## 不粘锅特氟龙涂层

### 安全使用温度

```
特氟龙（聚四氟乙烯/PTFE）熔点：327℃
  → 空烧超过260℃ → 涂层分解
  → 产生含氟气体（聚合物烟雾）

症状：聚合物烟雾热病（Polymer fume fever）
  → 寒颤/发热/头痛
  → 通常24-48小时自愈

涂层损坏阈值：
  → 260-300℃持续加热 → 涂层开始剥落
  → 刮伤/用金属铲 → 涂层机械损伤
  → 剥落的涂层摄入 → 无已知毒性（惰性物质）
    但涂层完整性破坏 → 继续加热风险增加
```

### 使用建议

```
✅ 木铲/硅胶铲
✅ 中小火烹饪（油烟开始出现 = 温度过高）
✅ 涂层磨损明显 → 更换（通常2-3年）
✅ 不空烧
```

## 微波炉加热禁忌

### 绝对禁止加热

```
金属物品：
  → 金属反射微波 → 产生电弧（放电）→ 损坏微波炉
  → 铝箔（任何形状）→ 电弧 → 火灾风险

封闭/带壳食物：
  → 鸡蛋（带壳/去壳）→ 内部水分变蒸汽 → 爆炸
  → 封闭容器 → 压力积聚 → 爆炸
  → 葡萄/圣女果 → 内部液体变蒸汽 → 爆裂

塑料容器（无标识）：
  → 无"微波炉可用"标识 → 高温下释放塑化剂
  → 一次性饭盒 → 变形 + 有害物质析出

纸袋/报纸：
  → 可能含金属油墨 → 火灾风险
  → 干燥纸张 → 过热 → 燃烧
```

### 安全的容器材质

```
✅ 玻璃（所有类型）
✅ 陶瓷（无金属花纹）
✅ 硅胶（食品级）
✅ 纸（微波炉专用纸）
✅ 标注"微波炉可用"的塑料容器
```
