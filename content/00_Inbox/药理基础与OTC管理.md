# 💊 药理基础与OTC管理

## 抗生素 vs 消炎药：严格区分

### 本质区别

```
抗生素（Antibiotics）：
  → 杀菌/抑菌，针对**细菌感染**
  → 名称特征：XX霉素、XX西林、头孢XX、XX沙星
  → 对病毒感染无效（感冒/流感/新冠都是病毒）

消炎药（NSAIDs / 激素）：
  → 控制炎症反应，缓解红肿热痛
  → 对细菌感染无效，仅缓解症状
```

### 细菌 vs 病毒：如何区分

| 症状特征 | 细菌感染 | 病毒感染 |
|---------|---------|---------|
| 发热 | 持续高热（>38.5℃） | 反复发热，时高时低 |
| 痰/鼻涕 | 黄绿色脓性 | 清亮/白色 |
| 喉咙 | 扁桃体脓点 | 咽痛无脓点 |
| 病程 | 加重趋势，>10天不好转 | 3-5天达峰后缓解 |
| 白细胞 | 升高（中性粒细胞为主） | 正常或降低 |

> ⚠️ 感冒初期（1-3天）几乎都是病毒，不需要抗生素。抽血+医生听诊才能确诊细菌感染。

### 滥用抗生素的危害

```
短期：过敏反应、胃肠道紊乱
长期：耐药性产生 → 以后真正需要时无药可治
全球危机：抗生素耐药已被WHO列为公共卫生危机
```

## 退烧药：对乙酰氨基酚 vs 布洛芬

### 对比一览

| 特性 | 对乙酰氨基酚（泰诺/扑热息痛） | 布洛芬（芬必得/美林） |
|------|--------------------------|---------------------|
| 退热效果 | 中等 | 较强 |
| 止痛效果 | 头痛/关节痛 | 头痛/牙痛/肌肉痛/痛经 |
| 每次剂量 | 325-500mg | 200-400mg |
| 用药间隔 | ≥4小时 | ≥6小时 |
| 每日最大量 | 2000mg（成人） | 1200mg（成人） |
| 适合人群 | 孕妇/蚕豆病/胃不好的人 | 需同时消炎的人 |

### 交替使用禁忌

```
⚠️ 两者不可同时服用！
├── 退热效果不叠加
├── 肝肾负担加倍
└── 间隔：对乙酰氨基酚4h → 布洛芬6h（交替使用需间隔至少1-2h）

❌ 以下人群禁用布洛芬：
├── 胃溃疡/十二指肠溃疡
├── 严重心衰
├── 孕妇（孕晚期）
└── 服用抗凝药（华法林）的人
```

## 肝毒性药物叠加危险

### 常见肝损药物组合

```
高危组合1：感冒药 + 酒精
  ├── 感冒药（含对乙酰氨基酚）
  ├── 酒精（抑制肝脏代谢）
  └── 风险：急性肝坏死

高危组合2：多种感冒药叠加
  ├── 泰诺 + 白加黑 + 999感冒灵
  ├── 三者都含对乙酰氨基酚
  └── 极易超过每日最大安全剂量（2000mg）

高危组合3：他汀类降脂药 + 抗生素
  ├── 阿托伐他汀/辛伐他汀
  ├── 与大环内酯类抗生素（阿奇霉素/红霉素）
  └── 风险：横纹肌溶解症

高危组合4：中药"保肝" + 化学药
  ├── 何首乌等中药已知肝毒性
  └── 与其他肝代谢药物叠加
```

### 护肝原则

```
┌─────────────────────────────────────────┐
│  ① 吃药不喝酒，喝酒不吃药                 │
│  ② 不自行叠加多种感冒药/止痛药            │
│  ③ 长期服药者每3-6个月查肝功能            │
│  ④ 服用他汀每3个月查肌酸激酶（CK）        │
│  ⑤ 保健品/中药不乱吃，警惕不明成分        │
└─────────────────────────────────────────┘
```

## OTC管理：非处方药使用规范

### OTC判断

```
红色OTC（甲类）→ 只能在医院/药房购买
绿色OTC（乙类）→ 超市/便利店也可销售
处方药（Rx）→ 必须凭医生处方购买
```

### 家庭常备药箱建议

```
退热止痛：对乙酰氨基酚/布洛芬
感冒初期：含伪麻黄碱类（缓解鼻塞）
腹泻：蒙脱石散（吸附毒素）+ 口服补液盐（防脱水）
过敏：氯雷他定/西替利嗪
轻微烫伤：湿润烧伤膏（不是牙膏！）
创可贴：含碘棉片消毒后再贴
工具：体温计、无菌纱布、医用胶带
```
