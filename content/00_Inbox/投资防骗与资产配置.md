# 📈 投资防骗与资产配置

## 庞氏骗局的财务特征识别

### 核心特征

```
庞氏骗局 = 用新投资者的钱偿还老投资者的收益
        = 没有真实盈利来源，纯粹吃新还旧
```

### 识别红线（符合任意一条需高度警惕）

| 特征 | 说明 | 举例 |
|------|------|------|
| 承诺保本+高收益 | "稳赚不赔"、"年化20%+" | P2P、原始股骗局 |
| 收益来源不明 | 说不清钱投向了哪里 | 传销币、资金盘 |
| 需要发展下线 | 金字塔结构，收益靠拉人 | 传销、层压式推销 |
| 提现有冷却期 | 不能随时退出，拖延跑路时间 | 多地非吸平台 |
| 监管查无资质 | 无金融牌照，无备案 | 野鸡理财公司 |

> **记住**：所有超过无风险收益率（国债收益率约2-3%）的"稳定"收益，都在用风险换钱。

## 复利效应与七二法则

### 复利的力量

> **72法则**：本金翻倍时间（年）≈ 72 ÷ 年化收益率（%）

| 年化收益率 | 翻倍所需年数 |
|-----------|------------|
| 4% | 18年 |
| 6% | 12年 |
| 8% | 9年 |
| 12% | 6年 |
| 20% | 3.6年 |

**复利的关键**：时间 × 收益率。正收益+时间=财富积累。

### 复利的反面：负复利

```
亏损10% → 需要涨11%才能回本
亏损50% → 需要涨100%才能回本
亏损90% → 需要涨900%才能回本
→ 亏损比盈利难太多，保住本金是第一原则
```

## 资产配置的"不可能三角"

```
收益性（Return）    ←→    安全性（Safety）
        \                    /
         \                  /
          ── 流动性 ──
        （Liquidity）
```

| 资产类型 | 收益性 | 安全性 | 流动性 |
|---------|--------|--------|--------|
| 银行存款 | 低 | 高 | 高 |
| 国债 | 低-中 | 极高 | 中 |
| 货币基金 | 中 | 高 | 高 |
| 指数基金 | 中-高 | 中 | 高 |
| 房产 | 高 | 中 | 低 |
| P2P/非吸 | 高 | 极低 | 低-中 |

> 💡 **现实解法**：不存在完美资产。用组合配置，在三者之间取平衡点。

### 经典组合公式（可调整）

```
┌──────────────────────────────────────────────────┐
│  30% 现金/货币基金  → 流动性储备（紧急用）          │
│  40% 指数基金/债券  → 稳健增值（跑赢通胀）          │
│  20% 成长型资产     → 追求超额收益                │
│  10% 高风险博弈     → 亏损不心疼的小比例试错        │
└──────────────────────────────────────────────────┘
```

## 货币基金 vs 指数基金

### 货币基金

```
本质：聚集大家的钱，投向短期银行存款、国债逆回购
特点：几乎不会亏损（历史上极少发生）
年化收益：约 1.5-2%（略高于活期）
适合：紧急备用金存放、短期闲置资金
代表：余额宝、零钱通、银行T+0理财
```

### 指数基金

```
本质：复制某个指数（如沪深300），买一篮子股票
特点：不依赖基金经理主观判断，长期上涨
年化收益：历史平均约 8-10%（A股宽基）
适合：有3-5年以上投资周期，能承受波动的人
代表：沪深300ETF、中证500ETF、标普500ETF
```

**核心区别**：货币基金是"不会亏的零钱罐"，指数基金是"用时间换收益的长期仓"。

## 投资三原则

```
┌─────────────────────────────────────────────────┐
│  ① 不懂不投：看不透盈利模式就不投                │
│  ② 分散配置：单一资产不超过总资产的30%           │
│  ③ 用闲钱投资：3-5年不用的钱才进权益类市场        │
└─────────────────────────────────────────────────┘
```
