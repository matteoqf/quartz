# 🔧 家庭配电与水管维修

## 家用配电箱排障

### 配电箱结构

```
总开关（通常40-63A）
  ├── 照明回路（10-16A，空气开关）
  ├── 普通插座回路（16A，空气开关）
  ├── 大功率插座回路（20-25A，漏电保护器）
  ├── 空调回路（16-20A，漏电保护器）
  └── 厨房回路（20-25A，漏电保护器）
```

### 跳闸原因逆向排查

| 跳闸类型 | 原因 | 排查步骤 |
|---------|------|---------|
| 漏电跳闸（漏电保护器） | 某电器漏电/线路老化 | 拔掉所有电器 → 逐个插回 → 定位故障电器 |
| 过载跳闸（空开） | 同时使用大功率电器 | 计算各电器功率之和，看是否超载 |
| 短路跳闸 | 电线破皮/接线错误 | 通常有明显焦味，需要专业电工 |

### 漏电保护器测试

```
每月按一次"测试按钮"（T字按钮）
→ 跳闸 = 功能正常（每月测试一次）
→ 不跳闸 = 失效（立即更换）
```

## 角阀与上水软管更换周期

### 强制更换周期

```
上水软管（连接角阀和水龙头/热水器）：
  → 建议每3-5年更换
  →  braided不锈钢软管（编织管）也有使用寿命
  → 爆裂风险：老化的编织管在高压下可能破裂
  → 关闭总阀 → 拧下旧管 → 缠生料带 → 装新管

角阀（控制每个用水点的总开关）：
  → 发现滴水/锈蚀 → 立即更换
  → 更换时关总阀 → 角阀顺时针关闭
  → 滴水：内部阀芯磨损，需更换
```

### 存水弯防臭防虫原理

```
U型/S型存水弯工作原理：
  → 形成水封（water seal）
  → 弯管内存有一段水
  → 阻止下水道臭气/虫子通过
  → 干涸 = 失去防护（长期不用的地漏/洗手池）
  → 每周/每月往地漏倒一杯水
```
