# 🛡️ 刑事边界与自我防御

## 正当防卫与防卫过当

### 法律规定

```
《刑法》第20条第1款：
  为了使国家、公共利益、本人或他人的人身、财产
  免受正在进行的不法侵害，而采取的制止不法侵害的行为
  → 不负刑事责任（正当防卫）
```

### 正当防卫四要件

| 要件           | 内容                   | 举例                 |
| -------------- | ---------------------- | -------------------- |
| 不法侵害存在   | 客观存在非法攻击       | 有人持刀砍你         |
| 侵害正在进行   | 已经开始，尚未结束     | 刀已举起，尚未砍下   |
| 具有防卫意识   | 出于保护目的，非报复   | 反击是为了阻止侵害   |
| 未超过必要限度 | 防卫手段与侵害程度相当 | 对方徒手，你不能用刀 |

### 防卫过当的判定

```
超过必要限度造成重大损害 → 防卫过当
                          → 应当减轻或免除处罚

典型过当场景：
├── 对方只是轻微殴打，你捅了一刀
├── 侵害已经停止（对方跑了），你追上去打
└── 为保护财产，用致命手段对付非致命攻击
```

### 特殊防卫权

```
《刑法》第20条第3款 — 无过当防卫：
  对正在进行的行凶、杀人、抢劫、强奸、绑架
  等严重危及人身安全的暴力犯罪 → 无限防卫权
  打死打伤不负刑事责任
```

### 应对步骤

```
第一步：核实身份
  ├── 要求对方出示工作证件（姓名+警号/工号）
  ├── 记录对方单位、姓名、警号
  └── 不可抗拒的情况下才配合

第二步：行使沉默权
  ├── "我有权保持沉默"
  ├── 任何时候都可以不回答问题
  └── 回答不说谎（说谎可能构成伪证罪）

第三步：要求律师在场
  ├── "我要见我的律师"
  ├── 在律师到达之前不回答任何实质问题
  └── 没钱请律师 → 国家会为你指定（法律援助）

第四步：不签署任何你不理解的文件
  ├── 要求解释每一条款的含义
  ├── 拒绝在空白文件上签字
  └── 签署前必须拿到复印件
```

### 传唤 vs 拘传

| 区分     | 传唤                     | 拘传                 |
| -------- | ------------------------ | -------------------- |
| 强制程度 | 任意侦查措施（自愿到案） | 强制措施（强制带到） |
| 时限     | 一般不超过12小时         | 拘传后最长24小时     |
| 律师     | 可在询问前/中请求        | 同上                 |

## 帮信罪：日常防范

### 帮信罪是什么

> **帮助信息网络犯罪活动罪**（简称帮信罪）：明知他人利用信息网络实施犯罪，为其提供互联网接入、服务器托管、网络存储、通讯传输等技术支持，或提供广告推广、支付结算等帮助，情节严重。

### 常见"中招"场景

```
⚠️ 高危行为（绝对禁止）：
├── 出租/出借个人银行卡（含储蓄卡和信用卡）
├── 出租/出借个人手机卡、支付宝/微信支付账户
├── 帮人"刷脸"认证（人脸信息用于解封账户）
├── 为他人注册公司/办理营业执照并交给别人使用
├── 提供微信/支付宝收款码给他人转账（跑分）
└── 帮助陌生人"过账"（资金中转）
```

### 帮信罪的认定标准

```
情节严重（满足任一即构成）：
├── 为1个以上对象提供帮助
├── 支付结算金额 ≥ 20万元
├── 银行卡一张/手机卡三张以上
├── 违法所得 ≥ 1万元
└── 造成被害人人身/财产重大损失
```

### 防范口诀

```
"两卡"不借，"四件"不给：
  ❌ 银行卡  ❌ 手机卡
  ❌ 支付宝  ❌ 微信支付
  ❌ 身份证  ❌ 人脸认证

遇到"轻松赚快钱" → 100%是陷阱
```
