# 🏥 医疗系统逆向工程

## 医院科室分诊树图

```
发热/不明症状
  └── 首诊：普通内科 / 全科医学
        ├── 咳嗽+咳痰 → 呼吸内科
        ├── 腹痛+消化问题 → 消化内科
        ├── 心悸+胸闷 → 心血管内科
        ├── 头痛+头晕 → 神经内科
        ├── 尿频+尿痛 → 泌尿外科/肾内科
        └── 皮疹+过敏 → 皮肤科

外伤/骨折
  └── 急诊科 / 骨科

急性剧烈腹痛（阑尾炎/穿孔/宫外孕）
  └── 急诊科 → 急诊手术

儿科（<14岁）
  └── 儿童医院 / 儿科

妇科/产科
  └── 妇科（炎症/肿瘤）
  └── 产科（怀孕/分娩）

口腔问题
  └── 口腔科（牙/颌面）
```

### 急诊 vs 门诊

| 对比 | 急诊 | 门诊 |
|------|------|------|
| 收治 | 急性/危重/病情突变 | 稳定期/慢性/计划性就诊 |
| 排队 | 先分诊分级（危重优先） | 按预约/挂号顺序 |
| 费用 | 通常更高（含急诊诊疗费） | 相对较低 |
| 可选专家 | 通常轮转值班医生 | 可选特定专家 |
| 适合 | 胸痛/腹痛/大出血/意识障碍 | 复查/慢病/体检异常复查 |

## 医保三目录与报销机制

### 三目录

```
基本医疗保险目录（甲乙丙三类）：

甲类（国家统一制定）：
  → 全部纳入医保报销
  → 城镇职工报销约80-90%
  → 居民医保报销约50-70%

乙类（各省市调整）：
  → 部分自付（通常10-20%）
  → 剩余部分按比例报销

丙类（不在目录）：
  → 100%自费（进口药/新药/特药）
  → 通常是效果更好但价格昂贵的药
```

### 关键参数

```
起付线（门槛费）：
  → 超过起付线的费用才计入报销
  → 各地不同（三甲医院通常500-1000元）

封顶线：
  → 年度最高报销限额（通常20-50万）
  → 超过封顶线全部自费

报销比例：
  → 医院级别越高（社区医院 → 三甲），报销越低
  → 职工医保 > 居民医保/新农合
```

### 住院报销示例

```
三级医院，城镇职工医保：
  总费用 ¥30,000
  - 丙类药 ¥5,000（纯自费）
  - 乙类自付 ¥2,000（10%自付）
  - 起付线 ¥1,000

  可报销基数 = 30,000 - 5,000 - 2,000 - 1,000 = ¥22,000
  报销比例 = 80%

  实际报销 = 22,000 × 80% = ¥17,600
  个人实际负担 = 30,000 - 17,600 = ¥12,400
```

## 异地就医直接结算

### 适用人群

```
├── 退休异地安置（回老家养老）
├── 异地长期居住（在外地买房定居）
├── 常驻异地工作（公司外派）
└── 异地转诊（当地治不了，需转上级医院）
```

### 备案流程

```
1. 线上备案：
   ├── 国家医保服务平台APP
   ├── 微信/支付宝：医疗健康 → 医保 → 异地就医
   └── 官网：参保地医保局官网

2. 选定点医院：
   ├── 就医地的医保定点医院
   └── 查询：国家医保服务平台（微信/APP）

3. 持社保卡/医保码就诊
   → 出院时直接结算（只需付自付部分）
```
