# 💼 劳动争议与维权

## 试用期开除的法定条件与赔偿

### 试用期不是"免责期"

```
劳动合同法》第21条：
  试用期解除劳动合同，应当向劳动者说明理由
  不能随意开除，必须证明"不符合录用条件"
```

### 合法解除条件

```
用人单位证明：
├── 录用条件已公示（招聘时写明）
├── 考核标准已告知（书面确认）
└── 有具体考核结果（数据/记录支撑）

⚠️ 口头说"不合适"无效，必须有书面证据
```

### 违法解除赔偿

```
经济补偿的2倍 = 2N
  N = 工作年限 × 月工资
  违法解除 = 2N 赔偿金
```

---

## 经济补偿金：N、N+1、2N

### 适用场景

| 情形 | 补偿类型 | 公式 |
|------|---------|------|
| 协商解除（单位提出） | **N** | 工作年限 × 月工资 |
| 医疗期满不能工作 | **N+1** | N + 1个月工资 |
| 客观情况重大变化解除 | **N+1** | N + 1个月工资 |
| 违法解除 | **2N** | 2 × 工作年限 × 月工资 |
| 企业破产/裁员 | **N** | 工作年限 × 月工资 |

### N+1 的"1"是什么

> 1个月工资 = 代通知金，即未提前30天通知的替代补偿
> 如果提前30天书面通知，则只需给N

### 计算基数

```
月工资 = 离职前12个月平均工资（含奖金/津贴/加班费）
封顶：社会平均工资3倍（超过按3倍计算）
年限封顶：12年（高收入群体）
```

## 竞业限制与保密协议

### 竞业限制规则

```
适用对象：高级管理人员、高级技术人员、负有保密义务的人员
期限：最长2年（约定）
补偿：离职后按月支付，标准不低于当地最低工资
```

### 违反后果

```
劳动者违反 → 支付违约金 + 继续履行竞业限制义务
用人单位违反 → 劳动者可主张继续履行或支付经济补偿
```

### 保密协议 vs 竞业限制

| 对比 | 保密协议 | 竞业限制 |
|------|---------|---------|
| 是否必须支付补偿 | ❌ 无需支付 | ✅ 必须支付 |
| 期限 | 永久有效（秘密存在期间） | 最长2年 |
| 主体 | 全体员工（合理范围） | 特定人员 |

## 工伤认定三要素

### 认定标准

```
工伤 = 工作时间 + 工作场所 + 工作原因
      （三要素同时满足）
```

### 三种工伤情形

```
1. 典型工伤
   工作时间 + 工作场所 + 因工作原因受伤

2. 视同工伤（48小时内死亡）
   在工作时间和工作岗位，突发疾病48小时内死亡

3. 特殊情形（单位证明）
   ├── 上下班途中（非本人主责）的交通事故
   ├── 因工外出期间，由于工作原因受伤
   └── 维护公共利益受到伤害
```

### 不予认定工伤

```
├── 故意犯罪
├── 醉酒或吸毒
├── 自残或自杀
└── 上下班途中，本人主责的交通事故
```

## 被迫解除劳动合同

### 法定被迫情形（可立即解除，无需提前通知）

```
├── 未按约定提供劳动条件
├── 未及时足额支付劳动报酬
├── 未依法缴纳社保
├── 单位规章制度违反法律法规，损害劳动者权益
└── 劳动合同无效（欺诈/胁迫）
```

### 维权路径

```
1. 收集证据（工资条/社保记录/劳动合同/聊天记录）
2. 向单位书面提交《被迫解除劳动合同通知书》
3. 提起劳动仲裁（免费，15天内仲裁委要受理）
4. 可主张：经济补偿金(N) + 未结工资 + 未休年假折算
```
