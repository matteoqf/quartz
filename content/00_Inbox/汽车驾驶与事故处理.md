# 🚗 汽车驾驶与事故处理

## 仪表盘故障灯等级分类

### 三色故障灯

```
🔴 红色 = 危险/立即停车
  ├── 机油压力灯（机油泵压力不足，熄火可能拉缸）
  ├── 发动机故障灯（长亮 = 排放/传感器问题）
  ├── 制动系统灯（刹车故障，可能无刹车）
  ├── 安全气囊灯（碰撞时可能不弹开）
  ├── 水温报警（发动机过热，可能开锅）
  └── 转向助力灯（无助力，方向盘变重）

🟡 黄色 = 警告/尽快检修
  ├── 发动机故障灯（短暂亮起可继续开，长亮需检修）
  ├── 胎压监测灯（轮胎漏气，降至安全值）
  ├── 安全带提醒（乘客未系安全带）
  ├── 变速箱故障灯（换挡可能异常）
  └── ABS灯（防抱死系统故障，刹车仍可用但无ABS）

🟢 绿色 = 正常/功能开启
  ├── 示宽灯/雾灯
  ├── 定速巡航
  └── 自动启停
```

## 高速爆胎应对

### 正确操作

```
爆胎发生时的标准操作（绝对不能做的事：踩刹车/猛打方向盘）：

第一步：紧握方向盘
  → 双手9点和3点位置（不是10点和2点）
  → 爆胎瞬间车辆会强烈偏向一侧
  → 方向盘控制方向是第一优先

第二步：避免刹车
  → 踩刹车会导致车身剧烈失控
  → 利用发动机牵阻减速（逐渐降档）

第三步：降档点刹
  → 手动挡：逐级降挡，利用发动机制动
  → 自动挡：切换到低档位（L/S档）
  → 车速下降后，轻踩刹车减速

第四步：靠边停车
  → 车辆停稳后
  → 第一时间打开双闪灯
  → 150米外放置三角警示牌
  → 人员撤到护栏外

禁止行为：
  ❌ 踩死刹车（车辆失控）
  ❌ 猛打方向盘（侧翻）
  ❌ 原地换胎（后车追尾）
```

## 交通事故快处快赔

### 拍照取证标准

```
必须拍摄三组照片：

第一组：前后全景（4张）
  ├── 道路标线在画面中（证明事故地点）
  ├── 全貌可见（双方车辆完整入镜）
  └── 远近各一张

第二组：碰撞点特写（2-4张）
  ├── 碰撞位置清晰可见
  ├── 包含碰撞深度/面积
  └── 从不同角度拍摄

第三组：车牌和车型（1-2张）
  ├── 双方车牌特写（放大清晰）
  └── 车辆受损部位

关键要点：
  → 不要只拍碰撞点，全景照才能定责
  → 照片时间戳是证据（手机默认带）
  → 移车前拍照（移车后现场证据消失）
```

### 交通事故定责典型场景

| 情形 | 责任划分 |
|------|---------|
| 追尾 | 后车全责（前车急刹除外）|
| 转弯未让直行 | 转弯车全责/主责 |
| 变道影响原车道 | 变道车全责 |
| 掉头未让直行 | 掉头车全责 |
| 逆行 | 逆行车全责 |
| 开门导致事故 | 开门者全责 |
