# 🥩 食品安全与食材保存

## 冰箱各层温差与食材放置

### 冰箱温度分布

```
冰箱内部温度梯度（从上到下）：

冷藏室（0-4℃）：
  ├── 上层：4℃（最温暖）→ 剩菜/即食食品
  ├── 中层：3-4℃ → 乳制品/鸡蛋/调味品
  ├── 下层：0-2℃（最冷）→ 生肉/海鲜（密封！）
  └── 门架：5-8℃ → 饮料/调料（温度波动最大）

冷冻室（-18℃以下）：
  └── 适用于：肉类/海鲜/速冻食品（可保存3-12个月）
```

### 存放规则

```
生肉/海鲜 → 必须密封（单独保鲜袋/盒子）+ 放最下层
  → 防止血水滴落污染其他食品

鸡蛋 → 放在冰箱内部托架（不是门架）
  → 门架温度波动大，影响保鲜

叶菜类 → 不用洗，用纸巾包好 → 吸水 + 保干
  → 放冷藏中层，不要贴冰箱内壁（冻伤）

热带水果（香蕉/芒果）→ 不要放冰箱
  → 低温会冻伤（棕黑色斑点）
  → 阴凉处常温保存
```

## 肉毒杆菌：自制发酵食品风险

### 肉毒杆菌特性

```
肉毒杆菌（Clostridium botulinum）：
  → 厌氧细菌（不需要氧气生存）
  → 产生肉毒毒素（已知最强天然毒素）
  → 0.1微克即可致命

中毒症状：
  → 视物模糊/重影（最早症状）
  → 吞咽困难/呼吸困难
  → 对称性上行性瘫痪
  → 死亡率30-50%

常见来源：
  → 自制罐头（密封+厌氧+室温 = 完美繁殖条件）
  → 真空包装食品（保存不当）
  → 蜂蜜（含孢子，1岁以下婴儿禁食）
```

### 家庭自制食品规则

```
✅ 灭菌彻底：罐头高温高压灭菌（121℃，30分钟以上）
✅ 酸性环境抑制：pH<4.5可抑制肉毒杆菌
✅ 冷藏保存：自制发酵食品一旦开罐必须冷藏
✅ 不自制真空密封常温保存的肉类罐头
✅ 1岁以下婴儿绝对不喂蜂蜜

⚠️ 警惕信号：
  ├── 罐头膨胀/变形（发酵产气）
  ├── 打开时有异常气味
  └── 试吃有异味（不是正常发酵味）
```

## 保质期 vs 最佳赏味期

### 工业定义区别

```
保质期（货架期）：
  → 在规定储存条件下，保持食品品质的期限
  → 超过保质期 → 食品可能不安全，禁止销售
  → 临近保质期 → 品质下降，但仍可食用（需自行判断）

最佳赏味期（最佳食用期）：
  → 食品在此日期前风味/口感最佳
  → 超过赏味期 ≠ 不能吃（只是口感下降）
  → 常见于：饼干/巧克力/咖啡/茶叶

"此日期前最佳"（BBE）：
  → 英国/澳洲使用
  → 与保质期类似，但更侧重风味
```

### 常见食品保质期参考

| 食品 | 常温/阴凉 | 冷藏 |
|------|----------|------|
| 新鲜肉类 | ❌ 不行 | 2-3天 |
| 剩菜 | ❌ | 1-2天 |
| 鸡蛋（带壳）| ❌ | 冷藏30天 |
| 牛奶 | 阴凉处开封后24h | 7天 |
| 面包 | 2-3天 | 5-7天（口感下降）|
| 罐头（未开封） | 1-3年 | 长期 |
