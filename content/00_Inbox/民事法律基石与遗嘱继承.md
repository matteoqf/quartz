# ⚖️ 民事法律基石与遗嘱继承

## 诉讼时效制度

### 基本概念

> 诉讼时效：权利人向法院请求保护民事权利的**有效期限**。超过时效，权利本身不消灭，但胜诉权消灭。

### 常见时效一览

| 类型 | 时效 | 起算点 |
|------|------|--------|
| 一般债权 | **3年** | 知道/应当知道权利受损 |
| 人身伤害 | **3年** | 治疗终结/知道侵权人 |
| 合同违约 | **3年** | 债务到期日 |
| 房产纠纷 | **20年** | 权利被侵害之日 |
| 最长保护期 | **20年** | 权利被侵害之日（客观） |

### 超过时效的后果

```
超时效 → 丧失胜诉权
        → 对方自愿履行，不能反悔要回
        → 法院不主动适用（被告不提，法院不问）
        → 仍有起诉权，但大概率输
```

## 定金与订金的法律效力差异

| 术语 | 法律性质 | 能否退还 | 超过20%部分 |
|------|---------|---------|------------|
| **定金** | 担保合同，违约定金罚则 | ❌ 违约方无权要求 | 可要求返还 |
| **订金** | 预付款，无担保效力 | ✅ 随时可要求退还 | 不存在 |

### 定金罚则

```
给付定金方违约 → 无权要求返还定金
收取定金方违约 → 双倍返还定金
```

### 常见场景

```
定金适用：购房合同、租车、装修合同
订金适用：网购预订、餐厅订位（无担保效力）
→ 签合同前看清楚写的是"定金"还是"订金"
```

## 夫妻共同财产与个人财产

### 婚姻存续期间财产划分

```
共同财产（默认共有）：
├── 工资、奖金、劳务报酬
├── 生产/经营/投资收益
├── 知识产权收益
└── 继承或受赠财产（无明确归一方）
    ❌ 彩礼归属有争议（多数法院认定为共同财产）

个人财产（永远私有）：
├── 婚前财产
├── 因人身伤害获得的赔偿/补偿
├── 遗嘱/赠与合同明确归一方的
└── 个人生活用品（首饰、衣物）
```

### 离婚时分割原则

```
协议离婚 → 双方协商（自由分割）
诉讼离婚 → 法院判决
         ├── 照顾子女/女方/无过错方原则
         ├── 债务共担（用于家庭生活的共同债务）
         └── 房屋：出资贡献 + 还贷情况 + 居住状况
```

## 遗嘱的法定形式与效力顺位

### 遗嘱的六种法定形式

| 形式 | 生效条件 | 特点 |
|------|---------|------|
| 自书遗嘱 | 亲笔书写+签名+日期 | 最方便，无需证人 |
| 代书遗嘱 | 2个见证人+代书人+签名 | 不会写字的人用 |
| 打印遗嘱 | 2见证人+每页签名 | 2021年后新增 |
| 录音录像遗嘱 | 2见证人+陈述内容 | 需明确身份 |
| 口头遗嘱 | 仅紧急情况+2见证人 | 危急消除后需重立 |
| 公证遗嘱 | 公证处办理 | 效力最强但可被撤销 |

### 效力顺位规则

```
遗嘱 > 遗赠扶养协议 > 法定继承
（同一人留有多份遗嘱 → 以最后一份为准，公证遗嘱除外）
```

### 遗嘱的无效情形

```
遗嘱全部/部分无效：
├── 无民事行为能力人或限制行为能力人所立
├── 伪造或被篡改的内容
├── 未为缺乏劳动能力又无生活来源的继承人保留必要份额
└── 处分了国家/集体/他人财产
```
