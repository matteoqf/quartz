# 汉语拼音平翘舌音（舌尖前音与舌尖后音）辨析表

---

## 1. 平舌音 (z, c, s)
平舌音发音时，舌尖抵住下齿背，气流从舌尖与上齿龈间的缝隙挤出。

| 声母 | 常用汉字举例 |
| :--- | :--- |
| **z** | 在、再、做、左、走、足、最、总、字、自、子、字、租、族 |
| **c** | 次、从、草、层、寸、词、辞、才、菜、参、采、仓、操、册 |
| **s** | 三、四、死、色、算、岁、虽、孙、俗、散、司、丝、私、送 |

---

## 2. 翘舌音 (zh, ch, sh, r)
翘舌音发音时，舌尖上翘，抵住或接近硬腭前部。

| 声母 | 常用汉字举例 |
| :--- | :--- |
| **zh** | 中、这、之、主、知、张、正、长、只、种、者、找、战、真 |
| **ch** | 出、吃、车、查、产、吹、唱、常、场、成、程、持、赤、初 |
| **sh** | 是、说、书、上、少、水、手、山、师、失、十、世、受、数 |
| **r** | 日、人、入、如、若、柔、热、容、让、肉、燃、染、儒、瑞 |

---

## 3. 易混淆对比组
通过对比练习可以更有效地掌握发音部位的转换。

* **z - zh**：自在 (zì zài) — 制止 (zhì zhǐ)
* **c - ch**：从此 (cóng cǐ) — 充斥 (chōng chì)
* **s - sh**：松散 (sōng sǎn) — 山水 (shān shuǐ)
* **混合练习**：
    * 四十 (sì shí)
    * 十四 (shí sì)
    * 真正 (zhēn zhèng)
    * 层次 (céng cì)

---

> **辨析技巧提示**：
> 1. **记少不记多**：在普通话中，翘舌音的字数远多于平舌音，重点记忆平舌音字能提高效率。
> 2. **声旁类推**：利用汉字的形旁和声旁规律，如同声旁的字通常具有相同的平翘舌属性（如“仓”是平舌，“枪、抢”也是平舌）。
