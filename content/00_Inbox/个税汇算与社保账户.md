# 📋 个税综合所得汇算清缴与社保

## 综合所得税原理

### 税制结构

```
收入类型
├── 工资薪金（最常见，税率3-45%累进）
├── 劳务报酬（税率20-40%）
├── 稿酬（减征30%后按20%计）
└── 特许权使用费（20%固定）
         ↓ 合并计入综合所得
   年度汇算清缴（3月1日-6月30日）
         ↓ 减除6万免征额 + 专项扣除
   应纳税所得额 × 税率 - 速算扣除数
```

### 税率表（2024年起）

| 级数 | 应纳税所得额 | 税率 | 速算扣除数 |
|------|------------|------|----------|
| 1 | ≤ 36,000 | 3% | 0 |
| 2 | 36,000-144,000 | 10% | 2,520 |
| 3 | 144,000-300,000 | 20% | 16,920 |
| 4 | 300,000-420,000 | 25% | 31,920 |
| 5 | 420,000-660,000 | 30% | 52,920 |
| 6 | 660,000-960,000 | 35% | 85,920 |
| 7 | > 960,000 | 45% | 181,920 |

## 专项附加扣除（合法减税神器）

### 七大扣除项

| 扣除项 | 每月扣除标准 | 年度可减税（约） |
|--------|------------|----------------|
| 子女教育 | 2,000元/子女 | 1,440-5,400元 |
| 继续教育 | 400-3600元/月 | 480-3,600元 |
| 大病医疗 | 8万限额据实 | 最高8,000元 |
| 住房贷款利息 | 1,000元/月 | 1,200元 |
| 住房租金 | 800-1,500元/月 | 960-1,800元 |
| 赡养老人 | 2,000-3,000元/月 | 2,400-3,600元 |
| 3岁以下婴幼儿照护 | 2,000元/子女 | 1,440-5,400元 |

> 💡 **顶格申报**：每项都如实申报，不要漏填。漏填=主动多交税。

## 五险一金全解析

### 缴费比例（各地略有差异）

```
┌──────────────┬──────────────┬──────────────┐
│    险种      │   个人缴纳    │   单位缴纳    │
├──────────────┼──────────────┼──────────────┤
│ 养老保险      │    8%        │    16%       │
│ 医疗保险      │    2%        │    8-10%     │
│ 失业保险      │    0.5%      │    0.5%      │
│ 工伤保险      │    0%        │    0.2-1.4%  │
│ 生育保险      │    0%        │    0.5-1%    │
│ 住房公积金    │   5-12%      │   5-12%      │

```

### 统筹账户 vs 个人账户

| 账户 | 归属 | 用途 | 继承性 |
|------|------|------|--------|
| 统筹账户 | 社会公用 | 基础养老金发放、医保报销 | ❌ 不继承 |
| 个人账户 | 你自己 | 养老金计算基数、医保余额 | ✅ 可继承 |

## 养老金跨省转移接续

### 为什么要转移

多地参保 → 退休地只选一个 → 需合并缴费年限

### 转移条件

- 男性 < 50周岁，女性 < 40周岁：可随时转
- 男性 ≥ 50，女性 ≥ 40：**只能临时账户**，退休时再归集

### 转移步骤

```
1. 在新参保地参保后，申请开具《基本养老保险参保缴费凭证》
2. 向原参保地提出转出申请
3. 原参保地寄送《信息表》给新参保地（15个工作日内）
4. 新参保地办理转入确认
```
