# ✈️ 公共交通与航空安全

## 飞机失压：氧气面罩佩戴次序

### 失压时的正确操作

```
失压（舱内气压骤降）：
  → 飞机高度>3000米时，大气压急剧下降
  → 缺氧会在15-20秒内导致失去意识

正确顺序：
  ① 坐好系好安全带
  ② 座位上方的氧气面罩会自动掉落（用力拉）
  ③ 先给自己戴好面罩（先己）
     → 戴好后，用一只手压住面罩
     → 确保密闭
  ④ 再为儿童/同行者戴好（后人或同时）
     → 如果你先昏迷，机上没人能帮你

为什么先自己？
  → 缺氧后你无法帮助任何人
  → 儿童有更长的无意识生存时间（代谢率低）
  → 自己先戴好 = 保持清醒能力 = 之后可以救人
```

## 火车/轨交紧急制动阀

### 合法使用场景

```
紧急制动阀（俗称"太平锤"）位于车厢两端
  → 使用后列车会紧急停车

⚠️ 非紧急情况使用：
  → 违法（《治安管理处罚法》）
  → 可能造成乘客摔倒/受伤
  → 可能引发二次事故（后车追尾）

✅ 紧急情况（可以使用）：
  ├── 乘客突发疾病/心脏骤停
  ├── 乘客被夹在站台与列车之间
  ├── 发现火情/烟雾
  └── 发现重大安全隐患

使用方法：
  → 打开玻璃罩
  → 用力拉下手柄/旋转
  → 等待列车停稳后听从工作人员指挥
```

## 踩踏前的洋流效应识别

### 人群动力学

```
踩踏事故的成因：

"洋流效应"：
  → 人群像潮水一样移动
  → 当密度超过每平方米4-5人时
  → 个人失去自主行动能力
  → 被人群裹挟移动

⚠️ 危险信号（发现即远离）：
  ├── 人群突然从不同方向涌来（汇聚点）
  ├── 前面的人开始被推搡但无法移动
  ├── 脚离地面的情况出现（被人抬起）
  ├── 周围人开始尖叫/恐慌
  └── 出现"压力墙"（人群像一堵墙压过来）
```

### 自保姿势

```
已经身陷人群时：

双臂胸前防压迫姿势：
  ├── 双手握拳，肘关节撑开
  ├── 肘部抵住前面人的肩膀/胸部
  ├── 保持胸部呼吸空间
  └── 随人群移动，不要逆行

被挤倒时：
  ├── 双手护住头部/颈部（蜷缩成球）
  ├── 侧卧（不完全仰躺）
  ├── 膝盖蜷缩保护腹部
  └── 等待人群散去后起身
```

## 领事保护与协助边界

### 领事能做什么 vs 不能做什么

```
✅ 领事可以做的：
  ├── 在海外中国公民合法权益受到侵害时，交涉驻在国政府
  ├── 推荐律师/翻译/医疗机构（不提供资金）
  ├── 协助联系国内亲属
  ├── 补发/换发旅行证件
  ├── 因客观原因无法汇款时，协助向国内家属汇款
  └── 在发生重大灾害时，协助撤离

❌ 领事不能做的：
  ├── 干预驻在国司法/行政程序
  ├── 介入商业纠纷（债务/合同）
  ├── 介入劳资纠纷
  ├── 为中国公民提供仲裁/审判结果
  ├── 代为支付律师费/医疗费
  ├── 帮助你违反当地法律（违章/违法同样追责）
  └── 干预驻在国的内政
```
