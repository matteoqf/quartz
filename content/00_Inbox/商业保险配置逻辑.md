# 🛡️ 商业保险配置逻辑

## 社保与商保的底层互补关系

```
社保（强制保险）         商保（自主选择）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
广覆盖，低保障           精准覆盖，高保障
不盈不亏，劫富济贫       商业精算，对冲风险
基础养老金替代率 ~40%    补充养老金缺口
医保报销有上限/目录限制   突破目录，覆盖自费药
```

> 社保是**打底**，商保是**加固**。两者不是二选一，而是叠加。

---

## 四大险种对比

| 险种 | 核心功能 | 适用场景 | 优先级 |
|------|---------|---------|--------|
| 医疗险 | 报销医疗费用 | 所有人 | ⭐⭐⭐ 最高 |
| 重疾险 | 一次性给付 | 家庭经济支柱 | ⭐⭐⭐ 最高 |
| 意外险 | 意外伤残/死亡 | 所有人 | ⭐⭐ 高 |
| 定期寿险 | 身故/全残赔付 | 有负债/抚养义务 | ⭐⭐ 高 |

## 医疗险详解

### 百万医疗（主流产品）

```
保障特点：
├── 保额：通常 200-600万
├── 免赔额：通常 1万元（社保报销后）
├── 突破社保目录：覆盖自费药、进口器材
└── 优点：保费低（30岁约300元/年），保额高
```

**适合人群**：健康人群作为医保补充，覆盖大额医疗风险

### 医疗险核心条款辨析

| 条款 | 说明 | 避坑要点 |
|------|------|---------|
| 免赔额 | 超过部分才报销 | 越低越好，但保费增加 |
| 报销比例 | 100% vs 60% | 优先选择100% |
| 院外购药 | 是否覆盖 | 癌症特药很贵，必须有 |
| 续保条件 | 停售/理赔后能否续 | 保证续保20年最优 |
| 就医范围 | 是否支持特需部/国际部 | 高端医疗可选 |

## 重疾险

### 核心逻辑

> 患约定重疾 → 一次性获得保险金（不问用途）→ 用于收入补偿、康复理疗、护理开销

### 赔付标准（28种重疾+3种轻症，统一行业定义）

```
重疾：确诊即赔（癌症）/ 手术完成后赔（心脏搭桥）
     / 达到某种状态（脑中风后遗症180天后）
轻症：通常是重疾早期/轻度状态
```

### 配置建议

| 年龄 | 建议保额 | 保障期限 | 理由 |
|------|---------|---------|------|
| < 30岁 | 3-5倍年收入 | 终身 | 保费低，保障时间长 |
| 30-50岁 | 3-5倍年收入 | 至70岁+ | 家庭责任最重阶段 |
| > 50岁 | 1-2倍年收入 | 至70岁 | 保费倒挂风险高 |

---

## 意外险

### 保障责任

```
意外险核心保障：
├── 意外身故：一次性给付
├── 意外伤残：按《伤残评定》1-10级比例赔付
└── 意外医疗：门诊/住院费用报销
```

**重要**：伤残按等级赔付，不是"全赔"或"不赔"。

### 特别注意

- ❌ 不保猝死（猝死是疾病，不是意外）
- ✅ 含猝死责任的意外险：需特别留意（加了猝死条款）
- ✅ 综合意外险 > 特定交通意外险

---

## 定期寿险

### 核心逻辑

> 家庭经济支柱身故/全残 → 保险公司赔钱 → 用于偿还房贷、抚养子女、赡养父母

### 配置要点

| 参数 | 建议 |
|------|------|
| 保额 | 10倍年收入 + 未还负债 |
| 保障期 | 覆盖家庭责任最重期（通常到子女独立/房贷还完） |
| 保障期 | 通常选20年/30年/至60岁 |

---

## 两大规则：犹豫期与等待期

```
犹豫期（通常15天）
  └── 签收保单后15天内可全额退保（零损失）
  └── 相当于"反悔期"

等待期（30-180天不等）
  └── 等待期内出险，不赔付（防止带病投保）
  └── 医疗险通常30-90天，重疾险通常90-180天
  └── 等待期内体检异常 ≠ 既往症（注意健康告知时机）
```

## 保险配置优先级路线图

```
第一层（刚需）：
  ✅ 医保 + 百万医疗险
  ✅ 意外险

第二层（家庭支柱必备）：
  ✅ 重疾险（保额要够）
  ✅ 定期寿险（家庭责任期）

第三层（有余力）：
  ✅ 增额终身寿险（财富传承/养老）
  ✅ 高端医疗（特需/海外就医）
```
