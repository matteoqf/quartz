# 🏦 个人征信与信用卡机制

## 征信报告构成与解读

### 五大模块

| 模块 | 内容 | 影响 |
|------|------|------|
| 个人基本信息 | 姓名、学历、婚姻、配偶 | 部分贷款审批参考 |
| 信贷交易记录 | 贷款、信用卡使用情况 | **核心权重** |
| 公共信息 | 社保、住房公积金、税务 | 辅助验证收入稳定性 |
| 异议标注 | 错误/待核实记录 | 申诉处理中 |
| 查询记录 | 机构查询次数与原因 | 频繁查询=资金饥渴 |

### 关键指标

```
✅ 优质信号：
  - 信用卡使用率 < 30%（总额度10万，用了3万）
  - 从不逾期
  - 贷款类型多样（房贷/车贷/消费贷组合）
  ⚠️ 警告信号：
  - 使用率 > 80%
  - 6个月内查询 > 4次（频繁申卡/申贷）
  - 有"代偿"记录（别人替你还债，说明你已违约）
```

## 信用卡核心机制

### 还款日历

```
账单日（固定某日，如每月5日）
    ↓ 生成账单（本期应付金额）
还款日（账单日后15-20天，如每月20日）
    ↓ 逾期
产生逾期利息 + 影响征信
```

### 利息陷阱：最低还款的复利深渊

> **最低还款额通常 = 应还金额的 5-10%**
> 但：利息按**全额**（不是剩余本金）计算，日息万分之五

**复利计算示例**：

| 情形 | 应付金额 | 选择最低还款(10%) | 实际年化利率 |
|------|---------|-----------------|------------|
| 消费 ¥10,000 | ¥10,000 | 首月还 ¥1,000 | **约 20%** |
| 全额逾期 | ¥10,000 | ¥1,000/月 | **约 20%** |

> ⚠️ 最低还款是银行最赚钱的业务，不到万不得已不要用。

### 分期付款的真实成本

| 宣传利率 | 实际年化（粗估） |
|---------|---------------|
| 0 手续费 | 实际约 10-15%（资金占用成本） |
| 3.5%/期 | 实际约 13-15% |
| 7.2%/期 | 实际约 15-18% |

## 贷款LPR利率转换逻辑

### LPR 是什么

**LPR（贷款市场报价利率）**：由18家银行共同报价形成的基准利率，每月20日公布。

### 转换公式

> **新利率 = LPR + 加点（可为负）**
> **加点 = 原合同利率 - 2019年12月LPR（4.8%）**

### 转换决策

| 原利率类型 | 转换建议 | 理由 |
|-----------|---------|------|
| 原利率 < 当前LPR | 选固定利率 | 已经享受优惠，避免未来上涨 |
| 原利率 > 5.5% | 选LPR浮动 | 大概率长期降息趋势 |
| 原利率 4-5% | 均可 | 差异不大 |

## 信用守护红线

```
🚫 绝对禁区：
  - 逾期90天以上（进入"黑名单"）
  - 连三累六（连续3次逾期/累计6次）
  - 信用卡大额套现（被识别直接降额封卡）
  - 为他人担保贷款（那人违约=你代偿）
```
