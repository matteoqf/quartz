# 🚨 院前急救与黄金窗口

## 心肺复苏（CPR）

### 什么情况需要CPR

```
心脏骤停 → 心脏失去泵血功能
         → 大脑缺血4-6分钟开始不可逆损伤
         → 超过10分钟 → 脑死亡
         → 必须立即进行胸外按压+人工呼吸
```

### CPR标准操作（成人）

```
1. 确认现场安全
2. 轻拍双肩 + 大声呼叫，确认无反应
3. 呼叫120（打开免提，边抢救边呼救）
4. 立即胸外按压：
   ├── 位置：两乳头连线中点（胸骨下半段）
   ├── 深度：5-6厘米（至少5cm）
   ├── 频率：100-120次/分钟
   ├── 手法：双手叠扣，肘关节伸直
   └── 姿势：跪在患者旁边，上半身重量压下去
5. 30次按压 → 2次人工呼吸（可选，非必须）
6. 持续按压至120/ AED/专业人员到达
```

### CPR关键参数

| 参数 | 成人 | 儿童（1-8岁） | 婴儿（<1岁） |
|------|------|------------|------------|
| 按压深度 | 5-6 cm | 约5 cm | 约4 cm |
| 按压频率 | 100-120次/分 | 100-120次/分 | 100-120次/分 |
| 按压与呼吸比 | 30:2 | 30:2 | 15:2 |
| 人工呼吸量 | 胸廓起伏 | 胸廓起伏 | 胸廓明显起伏 |

## AED（自动体外除颤器）

### 黄金救援时间

```
心脏骤停：
  1分钟内使用AED → 存活率 ~90%
  4分钟内使用AED → 存活率 ~50%
  每延迟1分钟 → 存活率下降7-10%
```

### AED操作四步骤

```
┌─────────────────────────────────────┐
│  ① 打开电源（AED会自动语音提示）      │
│  ② 贴电极贴（一张右上胸，一张左下胸）  │
│  ③ 插入电极插头（如需要）              │
│  ④ 按下放电键（确保无人触碰患者）      │
└─────────────────────────────────────┘

放电后 → 继续2分钟CPR → AED分析心律 → 必要时再次放电
```

### AED寻找技巧

```
常见配备场所：
├── 地铁站/火车站/机场
├── 大型商场/超市
├── 体育馆/游泳馆
├── 学校/医院
└── 高档写字楼/酒店大堂

手机地图搜索：AED（会自动显示附近位置）
```

## 海姆立克急救法

### 气道异物梗阻的识别

```
典型表现：
├── 无法说话、哭喊
├── 手捏喉咙（V字手势）
├── 面色青紫/发绀
└── 无法咳嗽（无法呼吸）
```

### 成人立位腹部冲击法

```
1. 站在患者背后，双手环抱腰部
2. 一手握拳，拇指顶住肚脐上方两横指处
3. 另一只手握住此拳
4. 快速向内向上冲击（每秒1次）
5. 重复至异物排出或患者失去意识
```

### 特殊人群处理

| 人群 | 方法 |
|------|------|
| 成人（卧位） | 骑跨在患者髋部，双手冲击 |
| 孕妇/肥胖者 | 胸部冲击法（位置在胸骨下半段） |
| 1-8岁儿童 | 背部拍背 + 腹部冲击（力道减半） |
| 婴儿（<1岁） | 5次背部拍背 + 5次胸部冲击（食指中指按压） |

## 烧烫伤的"冲脱泡盖送"

```
冲：第一时间用流动凉水冲15-30分钟（不要冰敷）
脱：小心脱去伤处衣物，粘连处剪开不要硬撕
泡：浸泡在凉水中15-30分钟
盖：用干净纱布/毛巾覆盖伤处
送：送医院（面积>10%、面部/关节/手足/会阴部烫伤必送）

❌ 绝对禁止：
├── 冰敷（会冻伤组织）
├── 涂牙膏/酱油/紫药水（加重感染风险）
├── 刺破水泡（破坏天然保护层）
└── 撕脱粘连衣物（可能连皮肤一起撕下）
```

## 中风"FAST"快速识别

```
F = Face（面部）
  → 请对方微笑，观察是否嘴角歪斜
  → 一侧面部下垂

A = Arm（手臂）
  → 请对方双臂平举，观察是否一侧无力下垂
  → 抬起后快速下落

S = Speech（语言）
  → 请对方重复简单句子，观察是否含糊/无法说话

T = Time（时间）
  → 上述任何一项出现 → 立即拨打120
  → 脑梗死4.5小时内可溶栓（黄金窗口期）
  → 记录症状出现时间（告诉医生）
```
