# 🧠 社会心理学与人际边界网

## 投射效应与煤气灯操纵识别

### 投射效应

```
投射效应：
  → 把自己的情绪/动机/想法归因到别人身上
  → "我觉得你讨厌我"（其实是你讨厌对方）

典型场景：
  ├── 职场PUA：老板把自己的焦虑投射给员工
  │    "你能力不行"（其实是老板自己决策失误）
  ├── 家庭：父母把自己的遗憾投射给孩子
  │    "我当年没机会，你必须考上"（孩子的压力）
  └── 亲密关系：把自己的不安全感投射为"对方不爱我"
```

### 煤气灯操纵（Gaslighting）

```
Gaslighting定义：
  → 通过系统性否定/歪曲事实
  → 让受害者怀疑自己的记忆/感知/理智
  → 最终受害者依赖操纵者来判断现实

常见手法：
  ① 否认事实（"我从来没说过这话"）
  ② 扭曲记忆（"你想太多了"）
  ③ 转移话题（质疑你时反过来攻击你）
  ④ 小恩小惠+持续否定（时而对你好时而贬低）
  ⑤ 引入第三方施压（让朋友说你敏感）

识别清单：
  ├── 对方经常说"你记错了吧"
  ├── 对方说"我都是为你好"
  ├── 你经常需要为对方的情绪道歉
  ├── 你越来越不确定什么是对/错
  └── 你的朋友/家人开始担心你
```

## 非暴力沟通（NVC）四要素

### 公式

```
NVC四要素：

  观察（Observation）→ 不评价的事实
    ✗ "你总是迟到"
    ✓ "这周你迟到了3次"

  感受（Feeling）→ 你的情绪反应
    ✗ "我觉得你不在乎我"
    ✓ "被冷落时我感到失落/焦虑"

  需要（Need）→ 背后的需求
    ✗ "你应该多陪陪我"
    ✓ "我需要感到被重视和陪伴"

  请求（Request）→ 具体的正向请求
    ✗ "以后不能这样了"
    ✓ "能否每周安排一次两个人一起吃饭的时间？"
```

### 练习模板

```
当你说/听到"你从不..."的时候：

第一步：把"从不/总是"替换成"有时候/这次"
  "你从不理解我" → "有时候我感觉你没理解我"

第二步：区分观察和评价
  观察 = 客观事实（具体事件）
  评价 = 主观判断（道德标签）

第三步：说出具体感受 + 需求 + 请求
```

## 课题分离理论

### 概念来源

```
阿德勒心理学核心概念：
  → 每个人都只能控制自己的课题
  → 他人的反应是他人的课题

判断标准（谁承担后果）：
  → 后果由谁承担 = 这是谁的课题
  → 你承担 = 你的课题
  → 他人承担 = 他人的课题
```

### 日常应用

```
场景1：同事拒绝帮忙
  你想说："帮我做一下这个表格"
  → 他的课题：是否帮忙（他承担后果）
  → 你的课题：如何表达需求（你承担后果）
  → 正确做法：表达请求，接受对方拒绝

场景2：父母催婚
  → 他们的课题：焦虑/期待（他们承担后果）
  → 你的课题：你自己的人生选择（你承担后果）
  → 正确做法：感谢关心，表达自己的节奏
  → 不需要为他们的焦虑负责

场景3：公开演讲紧张
  → 你的课题：做好准备，正常发挥
  → 听众的课题：是否喜欢你的演讲
  → 正确做法：专注在自己的表达，不要猜测每个人的反应
```

## 个人空间与公共距离

### 四类距离（Edward Hall）

```
亲密距离（Intimate）0-45cm
  → 父母/伴侣/子女
  → 正常社交中触碰这个距离会让人不适

个人距离（Personal）45-120cm
  → 朋友/同事
  → 正常交谈保持在45-75cm为舒适
  → 过近会让人产生防御

社交距离（Social）120-360cm
  → 正式场合/工作关系
  → 工作会议/商务谈判
  → 距离过近会被视为不礼貌

公共距离（Public）360cm以上
  → 演讲/表演
  → 公众人物与普通人之间的距离
```

### 东亚 vs 欧美的差异

```
东亚文化：
  ├── 公共空间个人距离偏近
  ├── 地铁/电梯里人与人距离较小
  └── 目光接触相对较少/较短

欧美文化：
  ├── 保持更大个人空间
  ├── 电梯里通常避免对视
  └── 目光接触代表自信/尊重

实际应用：
  → 在东亚城市不必刻意拉大距离
  → 欧美出差/旅行时保持更大距离
  → 地铁拥挤时：减少目光接触，减少身体接触
  → 电梯内：避免背对背站立时身体接触
```
